## 1. 主变压器安装工程 (Source: 7.pdf)
*适用场景：500kV/220kV 油浸式变压器/电抗器安装*

### 1.1 总体施工工艺流程
施工准备 → 本体就位检查 → 附件开箱检查及保管 → 套管及套管TA试验 → 附件安装及器身检查（含冷却器、油枕、套管） → 抽真空及真空注油 → 热油循环 → 二次接线 → 整体密封试验 → 变压器试验 → 结束。

### 1.2 关键工序作业方法（可直接引用）

#### 1. 变压器本体检查与内检
1.  **环境要求**：
    内检应在良好的天气下进行，防尘、防雨、防风措施应可靠。本体及附件温度应不低于周围环境温度。当空气相对湿度小于 75%时，器身暴露在空气中的时间不得超过 16h；调压开关在空气中暴露时间不超过 8h。
2.  **内检要点**：
    *   打开孔盖前，必须通过排气孔平衡油箱内外压力。
    *   检查铁芯及夹件的紧固情况，铁芯接地片与油箱的连接是否可靠，接地绝缘电阻符合要求（通常≥2MΩ）。
    *   检查绕组压板及垫块有无松动，引线绝缘包扎是否完好，引线夹持是否稳固。
    *   彻底清理箱底残油及杂质，确认无遗留物。

#### 2. 附件安装（套管、升高座、冷却器）
1.  **密封处理**：
    所有法兰连接处必须使用耐油橡胶密封垫或厂家指定的密封材料。密封垫压缩量应控制在厚度的 1/3 左右，做到密封良好，无渗漏。法兰连接螺栓应按对角线顺序均匀紧固，紧固力矩参考下表：
    *   M12螺栓：68 N·m
    *   M16螺栓：170 N·m
    *   M20螺栓：340 N·m
2.  **套管安装**：
    *   吊装套管前，应检查套管油位是否正常，瓷裙无损伤。
    *   套管吊装应使用专用吊具，保持倾斜角度与安装角度一致，防止碰撞瓷套。
    *   套管末屏接地必须可靠，安装后进行绝缘电阻测试。
3.  **冷却系统安装**：
    冷却器安装前应用合格的变压器油进行冲洗，直至流出的油与注入的油无差别。油泵、风扇电机的转向应正确，运行无异音、无振动。

#### 3. 抽真空与真空注油 (核心工艺)
1.  **抽真空要求**：
    *   注油前必须对变压器进行全真空处理。抽真空时应监视箱壁变形情况。
    *   **真空度标准**：真空度残压应 ≤13Pa (0.1 Torr)。
    *   **保持时间**：达到规定真空度后，500kV 变压器应保持真空度不少于 24h，方可开始注油。
2.  **真空注油流程**：
    *   注油应从油箱底部阀门注入，注油速度控制在 100L/min 以下，防止产生静电。
    *   注油全过程必须保持真空状态，注油油温宜控制在 40℃～60℃。
    *   注油至油箱顶部预留空间（距箱顶约 200mm）时停止，继续保持真空 4h 以上进行补油，直至规定的油面高度。
3.  **热油循环**：
    注油完毕后，进行热油循环。滤油机出口温度控制在 60℃～70℃，油箱内温度不低于 40℃。500kV 变压器热油循环时间应不少于 48h，且循环油量不少于 3 倍变压器总油量。

#### 4. 油务指标控制 (GB 50150-2016)
| 检验项目 | 500kV 标准值 | 220kV 标准值 |
| :--- | :--- | :--- |
| **击穿电压** | ≥ 60 kV | ≥ 40 kV |
| **微水含量** | ≤ 10 mg/L | ≤ 15 mg/L |
| **介质损耗 (90℃)** | ≤ 0.5 % | ≤ 0.5 % |
| **含气量** | ≤ 1.0 % (体积分数) | ≤ 1.0 % |

### 1.3 整体密封试验
变压器本体及附件全部安装完毕，注油至规定油位后，利用油枕或充气加压进行整体密封试验。
*   **试验压力**：油箱顶部施加 0.03MPa 压力。
*   **持续时间**：24h。
*   **合格标准**：无渗漏油，无永久变形。

## 2. 建筑电气与照明工程 (Source: 6.pdf)
*适用场景：主控楼、配电房、开关室的照明与动力配电*

### 2.1 施工工艺流程
安装准备 → 管路敷设/桥架安装 → 管内穿线/电缆敷设 → 灯具/开关/插座安装 → 配电箱安装 → 通电试运行。

### 2.2 主要施工方法

#### 1. 线管敷设
1.  **暗敷要求**：
    预埋在混凝土内的线管，其保护层厚度不应小于 15mm。消防管路保护层不小于 30mm。管路敷设应沿最近路线，减少弯曲。
2.  **弯曲半径**：
    线管弯曲半径不应小于管外径的 6 倍；埋设于地下或混凝土楼板内时，不应小于管外径的 10 倍。弯扁度不应大于管外径的 10%。
3.  **连接工艺**：
    镀锌钢管采用螺纹连接，连接处两端应用专用接地卡跨接接地线。PVC 管采用套管粘接，粘接牢固不漏浆。

#### 2. 管内穿线
1.  **色标规定**：
    严格执行相色规定：A相-黄色，B相-绿色，C相-红色，N线-淡蓝色，PE线-黄绿相间双色。
2.  **绝缘测试**：
    穿线前应清理管内积水杂物。穿线后，在连接开关、插座前，必须进行线路绝缘电阻测试。相间、相对地绝缘电阻值不应小于 0.5MΩ。

#### 3. 配电箱与终端安装
1.  **配电箱安装**：
    配电箱底边距地高度一般为 1.5m（壁挂式）。箱体垂直度允许偏差为 1.5‰。箱内接线应整齐、美观，导线余量适中，端子压接牢固，无松动。
2.  **开关插座**：
    *   同一室内开关/插座安装高度差不大于 5mm；并列安装高度差不大于 1mm。
    *   插座接线：单相两孔插座“左零右火”；单相三孔插座“左零右火上接地”。PE 线严禁串联连接。

## 3. 防雷与接地装置 (Source: 7.pdf, 6.pdf)

### 3.1 接地体施工
1.  **焊接要求**：
    接地体（线）的连接应采用焊接，焊接必须牢固无虚焊。搭接长度应符合规范：
    *   扁钢：搭接长度为宽度的 2 倍，且至少三面施焊。
    *   圆钢：搭接长度为直径的 6 倍，双面施焊。
2.  **防腐处理**：
    焊接部位应清除焊渣后，涂刷沥青漆或防腐涂料进行防腐处理。

### 3.2 设备接地
1.  **主变接地**：
    变压器本体应有两点可靠接地。铁芯接地电流测试应符合出厂值或标准要求（通常<100mA）。
2.  **等电位连接**：
    建筑物内的金属管道、构架、电缆金属外皮等均应与接地干线连接，形成等电位联结网络。
