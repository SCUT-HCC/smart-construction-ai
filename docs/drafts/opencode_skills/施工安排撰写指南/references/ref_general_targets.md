## 1. 质量管理目标 (Quality Targets)

### 1.1 通用质量目标 (适用于变电站/线路主体工程)
1.  **总体目标**：确保工程“零缺陷”投产，实现“达标投产，争创优质工程”。
2.  **分项指标**：
    *   土建分项工程合格率 100%，优良品率 ≥95%。
    *   电气安装分项工程合格率 100%，优良品率 ≥95%（部分核心设备要求 ≥96%）。
    *   工程建设期间不发生重大质量事故。
    *   工程资料及档案齐全、规范，符合达标投产要求。
    *   争创广东电网优质样板工程，争创省部级、国家级优质工程。

### 1.2 混凝土与基础工程专项质量目标
1.  **混凝土外观**：表面平整光洁，色泽一致，无蜂窝、麻面、露筋、孔洞及错台等外观缺陷。
2.  **基础偏差控制**：
    *   基础顶面标高偏差：≤5mm。
    *   基础轴线位移：≤10mm。
    *   预埋螺栓中心偏差：≤2mm；顶面标高偏差：0~+10mm。
3.  **原材料控制**：水泥、钢筋、砂石等原材料进场检验合格率 100%，见证取样送检率 100%。

### 1.3 钢结构与安装工程专项质量目标
1.  **焊接质量**：焊缝外观成型良好，无裂纹、夹渣、未熔合等缺陷。一、二级焊缝探伤检测合格率 100%。
2.  **防腐涂装**：涂层厚度符合设计要求（通常室外钢结构总厚度≥150μm），附着力良好，外观无流坠、气泡、脱皮。
3.  **安装精度**：构支架垂直度偏差 ≤H/1000，且不大于 10mm；梁面标高偏差 ±5mm。

## 2. 安全管理目标 (Safety Targets)

### 2.1 "六杜绝、三控制" 核心目标 (Source: 6.pdf, 4.pdf)
1.  **杜绝**发生人身死亡事故。
2.  **杜绝**发生人身重伤事故。
3.  **杜绝**发生重大机械设备损坏事故。
4.  **杜绝**发生重大火灾事故。
5.  **杜绝**发生负主要责任的交通死亡事故。
6.  **杜绝**发生重特大环境污染事故。
7.  **杜绝**发生集体食物中毒事故。
8.  **控制**轻伤负伤率在 3‰以下。
9.  **控制**年事故频率控制在 3‰以下。
10. **不发生**其他性质恶劣、不良社会影响的事故。

### 2.2 职业健康与作业环境目标
1.  作业人员体检率 100%，特种作业人员持证上岗率 100%。
2.  施工现场噪声、粉尘、有毒有害气体排放符合国家及行业标准。
3.  劳动防护用品配备率 100%，使用率 100%。
4.  无职业病发生。

## 3. 绿色施工与环保目标 (Green Construction Targets)

### 3.1 "四节一环保" 量化指标 (Source: 5.pdf)
| 指标类别 | 考核内容 | 计划目标值 |
| :--- | :--- | :--- |
| **环境保护** | 施工噪声控制 | 昼间 ≤70dB，夜间 ≤55dB (符合GB 12523) |
| | 扬尘控制 | 施工现场目测无扬尘，裸土覆盖率 100% |
| | 污水排放 | 生产生活污水处理排放达标率 100% |
| | 固体废弃物 | 建筑垃圾回收利用率 ≥30%，有毒有害废物分类处理率 100% |
| **节能与能源** | 临时用电 | 采用 LED 节能灯具占比 ≥80%，主要耗能设备采用变频技术 |
| **节材与材料** | 钢材/木材 | 钢材损耗率降低值 <1.5%，木模板周转次数 ≥5 次 |
| **节水与水资源** | 循环用水 | 现场设置雨水/基坑水收集系统，循环用水利用率 ≥30% |
| **节地与土地** | 临时设施 | 临时设施占地面积有效利用率 ≥90% |

### 3.2 现场环保具体措施承诺
1.  **扬尘治理**：施工现场主要道路进行硬底化处理，配备洒水车每天洒水降尘不少于 4 次。土方作业阶段配置雾炮机。
2.  **光污染控制**：夜间施工照明灯加设灯罩，透光方向集中在施工范围，避免直射周边居民区。电焊作业采取遮挡措施。
3.  **油污处理**：施工机械维修及保养产生的废油、含油棉纱等危险废物，设置专用危废收集桶，定期交由有资质单位处理，严禁随意倾倒。

## 4. 文明施工与标准化目标 (Civilized Construction)

### 4.1 南网 "7S" 管理目标 (Source: 16.pdf)
全面推行整理(Seiri)、整顿(Seiton)、清扫(Seiso)、清洁(Seiketsu)、素养(Shitsuke)、安全(Safety)、节约(Saving) 管理模式。
1.  **场地布置**：严格按照审批后的《施工总平面布置图》进行布置，功能分区合理（办公区、生活区、作业区、材料堆放区），标识清晰。
2.  **定置管理**：材料、设备按规划区域分类堆放整齐，挂设物资标识牌（名称、规格、产地、状态）。
3.  **工完料尽场地清**：每日作业完毕后，清理作业现场，废料及时回收，保持现场整洁。

### 4.2 可视化与标牌标准化
1.  **五牌一图**：在工地大门显著位置设置工程概况牌、管理人员名单及监督电话牌、消防保卫牌、安全生产牌、文明施工牌和施工现场平面布置图。
2.  **安全警示**：
    *   孔洞、临边设置刚性防护栏杆，并挂设“当心坠落”警示牌。
    *   高压设备区挂设“止步，高压危险”警示牌。
    *   起重作业区设置警戒线，并挂设“禁止入内”警示牌。
3.  **人员着装**：施工人员统一佩戴安全帽（管理人员红色/白色、施工人员蓝色/黄色，具体按项目部规定），穿着统一工作服，佩戴胸卡。
