## 1. 输电线路塔基基础施工 (Source: 1.pdf)
*适用场景：输电线路铁塔基础、人工挖孔桩*

### 1.1 挖孔桩施工工艺流程
测量分坑、场地清理 → 施工准备 → 孔口护壁制作 → 孔口安全防护（设置软梯、安全绳、通风管线） → 安装取土设备 → 坑内气体检测 → 开挖土方 → 护壁制作 → 挖至设计桩深 → 验孔 → 钢筋绑扎、支模 → 混凝土浇制及养护 → 拆模回填。

### 1.2 关键工序作业方法（直接引用）
1.  **土方开挖与护壁**：
    *   土方开挖由上而下逐层进行，每节开挖深度不得大于 1.0m。挖出的土方由专人提运出孔口，运至指定堆土场。
    *   为确保施工安全，成孔过程中必须设置护壁。第一节混凝土护壁应高出地面 100mm，并设置锁口，以阻挡井口周围土石杂物坠入孔内。
    *   护壁混凝土强度等级应符合设计要求（通常不低于C25），上下节护壁的搭接长度不得小于 50mm，且上下两节护壁竖向钢筋必须有效搭接。
2.  **通风与气体检测**：
    *   每日开工前及桩孔开挖过程中，必须使用毒气检测仪检测坑洞内有无毒害气体和缺氧现象，且每两个小时检测一次。
    *   当基础井挖孔深超过 10 米或孔内有沼气等有害气体时，应对孔内进行送风补氧，必须配备鼓风设备。宜用风机或风扇向孔内送风不少于 5min，排除孔内浑浊空气。
3.  **钢筋笼安装**：
    *   钢筋笼过长宜分段制作，入孔时再焊接。在搬运和安装过程中，应在笼内每隔 3～4 米装一个临时十字形加劲架，在钢筋笼安放入孔后拆除。
    *   钢筋笼的吊装应按规定吊点位置绑扎。清孔时，孔底沉渣或泥浆没有清理干净，造成实际孔深与设计要求不符，钢筋笼不到设计深度时，严禁强行下压。
4.  **混凝土浇筑**：
    *   **干式浇筑**：当孔底积水深度不超过 50mm 时，采用干式浇筑法。混凝土必须通过溜槽，当落距超过 3m 时，应采用串筒，串筒末端距孔底高度不宜大于 2m。
    *   **水下浇筑**：当积水较多且无法抽干时，应采用水下混凝土灌注法（导管法）。首批灌注混凝土的数量应能满足导管初次埋置深度和填充导管底部的需要。

## 2. 钢结构与屋面工程 (Source: 4.pdf)
*适用场景：变电站GIS室钢屋面、钢梁安装、压型钢板施工*

### 2.1 钢屋面施工工艺流程
施工准备 → 钢梁安装固定 → 纵向钢筋绑扎 → 混凝土泵送、浇筑 → 混凝土找平 → 混凝土养护。

### 2.2 压型钢板与栓钉施工
1.  **压型钢板安装**：
    *   压型钢板在钢梁上的支承长度不应小于 50mm。压型钢板侧向之间采用咬口钳连接。
    *   端部锚固：压型钢板端头与钢梁采用熔焊连接，焊点直径不小于 20mm，间距不大于 300mm。
    *   开孔处理：当楼板上开洞尺寸小于 500mm×500mm 时，开洞处不需采取加强措施，混凝土浇筑后用切割机切除；当开洞尺寸大于 500mm 时，应按设计要求加设洞口边梁。
2.  **栓钉（剪力钉）焊接**：
    *   栓钉焊接前，应将钢梁表面的油漆、铁锈等清除干净。
    *   焊接采用专用栓钉焊机。焊接后应进行外观检查，焊脚应均匀，焊缝表面无气孔、夹渣、裂纹等缺陷。
    *   **弯曲试验**：焊接完成后，应进行抽样弯曲试验，栓钉弯曲 30°后，焊缝处无裂纹为合格。

### 2.3 钢屋面混凝土浇筑
1.  **浇筑顺序**：
    混凝土浇筑应垂直于压型钢板波纹方向进行，避免混凝土堆积过高导致压型钢板变形。
2.  **振捣要求**：
    采用平板振动器或小直径插入式振捣器。振捣时不得直接接触压型钢板，以免造成钢板变形或漏浆。
3.  **平整度控制**：
    混凝土浇筑后，应及时用刮尺刮平，并用木抹子搓平。初凝前进行二次抹压，防止表面裂缝。

## 3. 防腐与防火涂料施工 (Source: 2.pdf, 3.pdf)
*适用场景：构支架防腐、钢梁防火保护*

### 3.1 表面处理 (关键工序)
1.  **除锈等级**：
    涂装前钢材表面必须进行除锈处理。设计要求除锈等级通常为 Sa2.5 级（喷砂/抛丸除锈），表面应无可见的油脂、污垢、氧化皮、铁锈和油漆涂层等附着物。
2.  **环境要求**：
    除锈后的钢材表面应在 4 小时内进行底漆涂装，防止重新生锈。当空气相对湿度大于 85%或钢材表面温度低于露点 3℃时，严禁进行除锈和涂装作业。

### 3.2 构支架防腐涂装
1.  **涂层结构**：
    通常采用“底漆+中间漆+面漆”配套体系。
    *   底漆：环氧富锌底漆，干膜厚度≥50μm。
    *   中间漆：环氧云铁中间漆，干膜厚度≥100μm。
    *   面漆：聚氨酯面漆或氟碳面漆，干膜厚度≥50μm。
2.  **涂装操作**：
    *   涂装宜采用高压无气喷涂法。喷枪移动速度应均匀，保持与工件表面垂直，喷嘴距离工件表面 250～350mm。
    *   上一道涂层干燥后，方可进行下一道涂层的涂装。涂层表面应平整、光滑、色泽一致，无流挂、起泡、剥落等缺陷。

### 3.3 钢结构防火涂料
1.  **基层处理**：
    防火涂料施工前，钢结构表面防锈底漆应涂装完毕并验收合格。清除表面的灰尘、油污。
2.  **喷涂作业**：
    *   防火涂料宜分层喷涂，每层喷涂厚度不宜超过 3mm（薄型）或 5-8mm（厚型）。
    *   第一遍喷涂不宜过厚，以盖住基层为准，防止流坠。待前一遍涂层表干后（约 4-24 小时），方可进行下一遍喷涂。
3.  **外观标准**：
    涂层应无裂纹（厚型涂料裂纹宽度不大于 1mm）、无空鼓、无脱落。涂层厚度应满足设计耐火极限的要求（如耐火 2.0h 对应厚度）。

## 4. 塔式起重机安拆 (Source: 13.pdf)
*适用场景：变电站大型设备吊装、特种设备管理*

### 4.1 塔吊安装流程
选定位置 → 浇筑承台基础（预埋螺栓） → 安装基础节 → 安装顶升套架 → 安装过渡节 → 安装回旋总成 → 安装平衡臂 → 安装起重臂 → 安装配重 → 电气接线、安装钢丝绳 → 塔机试运转 → 顶升塔身 → 整机调试。

### 4.2 顶升加节安全要点
1.  **平衡控制**：顶升过程中，必须调整塔机平衡，使塔机上部重心落在顶升油缸轴线上。严禁在顶升过程中进行回转、起升和变幅操作。
2.  **风力限制**：顶升作业严禁在四级以上大风中进行。
3.  **紧固检查**：每加完一节标准节，必须立即紧固连接螺栓。

### 4.3 塔吊拆除流程
拆卸平衡重（保留一块） → 拆卸起重臂 → 拆卸剩余平衡重 → 拆卸平衡臂 → 拆卸驾驶室及回转总成 → 拆卸标准节及附墙装置 → 拆卸底架及基础节。

### 4.4 应急处置（防台防风）
1.  **台风工况**：接到台风预警后，应将塔吊吊钩升至最高限位以下，小车收至臂根，大臂顺风停放，并处于自由回转状态。
2.  **断电检查**：切断塔机总电源，检查并紧固各连接螺栓，确保附墙装置可靠。
