## 3. 分项工程模板（6.2~6.9）

### 3.1 本体检查与内检（6.2）

#### 6.2.1 施工工艺流程

```
变压器到场 → 运输密封检查 → 取油样化验 → 绝缘电阻测量
→ 本体就位（起重机吊装）→ 核对方位/中心线
→ 平衡油箱内外压力 → 器身内检 → 封闭
```

#### 6.2.2 施工方法

1. 变压器到场后检查运输密封完好性，取油样送化验
2. 测量变压器绝缘电阻，确认合格
3. 使用 {{吨位}} 汽车起重机将本体就位至基础上，核对方位及中心线
4. 打开孔盖前必须通过排气孔平衡油箱内外压力
5. 检查铁芯及夹件紧固情况，铁芯接地绝缘电阻 ≥2MΩ
6. 检查绕组压板及垫块有无松动，引线绝缘是否完好
7. 彻底清理箱底残油及杂质

> 参数来源：electrical_install.md §1.2

#### 6.2.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 天气要求 | 良好，防尘防雨防风可靠 | — |
| 本体温度 | 不低于周围环境温度 | — |
| 相对湿度 <75% 器身暴露时间 | ≤16 h | GB 50150 |
| 调压开关暴露时间 | ≤8 h | GB 50150 |
| 铁芯接地绝缘电阻 | ≥2 MΩ | — |

> 参数来源：electrical_install.md §1.2

#### 6.2.4 质量控制要点

- 器身暴露时间严格计时，超时须重新干燥处理
- 铁芯绝缘电阻检测记录完整
- 箱底清洁度确认后方可封闭

#### 6.2.5 注意事项

- **跨章节一致性**：起重机吨位 → Ch5.3 设备清单

### 3.2 附件安装（6.3）

#### 6.3.1 施工工艺流程

```
附件开箱检查 → 密封垫准备 → 冷却器安装/冲洗
→ 油枕安装 → 套管安装（含绝缘测试）→ 螺栓紧固 → 密封检查
```

#### 6.3.2 施工方法

1. 密封材料采用耐油橡胶密封垫或厂家指定材料，压缩量为厚度的 1/3 左右
2. 螺栓按对角线顺序均匀紧固
3. 套管吊装前检查油位正常、瓷裙无损伤，使用专用吊具
4. 套管保持倾斜角度与安装角度一致
5. 套管末屏接地必须可靠，安装后做绝缘电阻测试
6. 冷却系统安装前用合格变压器油冲洗，直至流出油与注入油无差别
7. 油泵、风扇电机转向正确，运行无异音无振动

> 参数来源：electrical_install.md §1.3

#### 6.3.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 密封垫压缩量 | 厚度的 1/3 左右 | — |
| 螺栓紧固方式 | 对角线顺序均匀紧固 | — |
| M12 法兰螺栓力矩 | 68 N·m | — |
| M16 法兰螺栓力矩 | 170 N·m | — |
| M20 法兰螺栓力矩 | 340 N·m | — |

> 参数来源：electrical_install.md §1.3

#### 6.3.4 质量控制要点

- 所有法兰面密封垫更换为新品
- 套管末屏接地电阻实测
- 冷却系统油循环冲洗合格记录

#### 6.3.5 注意事项

- **跨章节一致性**：密封材料/变压器油 → Ch5.2 材料清单；专用吊具 → Ch5.3 设备清单

### 3.3 抽真空及真空注油（6.4）

#### 6.4.1 施工工艺流程

```
管路连接（按示意图）→ 启动真空泵 → 抽真空至残压达标
→ 保持真空 ≥24h（500kV）→ 底部阀门注油（保持真空）
→ 注油至距箱顶约 200mm 停止 → 保持真空 ≥4h → 补油至规定油面
```

#### 6.4.2 施工方法

1. 按抽真空管路示意图连接管路和布置机具，检查无误后操作
2. 开动真空泵抽真空，持续监视箱壁变形情况
3. 真空度残压达到 ≤13 Pa 后保持 ≥24h（500kV）
4. 注油口设在油箱底部阀门，注油速度 ≤100 L/min（防静电）
5. 注油全程保持真空状态，油温控制在 40℃~60℃
6. 注油至距箱顶约 200mm 时停止，保持真空 ≥4h 后补油至规定油面

> 参数来源：electrical_install.md §1.4

#### 6.4.3 关键技术参数

| 参数 | 标准值（500kV） | 标准值（220kV） | 依据 |
|------|---------------|---------------|------|
| 真空度残压 | ≤13 Pa (0.1 Torr) | ≤133 Pa | GB 50150 |
| 真空处理时间 | ≥36 h | ≥24 h | — |
| 真空保持时间 | ≥24 h | ≥24 h | — |
| 注油速度 | ≤100 L/min | ≤100 L/min | — |
| 注油温度 | 40℃~60℃ | 40℃~60℃ | — |
| 注油后保持真空 | ≥4 h | ≥4 h | — |

> 参数来源：electrical_install.md §1.4

#### 6.4.4 质量控制要点

- 真空度和保持时间严格记录
- 注油过程中持续监控油温和真空度
- 用透明软管连接顶部和底部阀作油位指示

#### 6.4.5 注意事项

- 此为**核心工艺**，对变压器绝缘性能影响最大
- **跨章节一致性**：真空泵、滤油机 → Ch5.3 设备清单；变压器油 → Ch5.2 材料清单

### 3.4 热油循环与静置（6.5）

#### 6.5.1 施工工艺流程

```
连接滤油机 → 启动热油循环 → 控制出口温度 60~70℃
→ 循环 ≥48h（500kV）→ 停泵 → 静置 ≥72h（500kV）
→ 取油样化验
```

#### 6.5.2 施工方法

1. 连接滤油机进行热油循环
2. 控制滤油机出口温度 60℃~70℃，油箱内最低温度 ≥40℃
3. 500kV 循环时间 ≥48h，循环油量 ≥3 倍变压器总油量
4. 循环结束后静置 ≥72h（500kV）/ ≥48h（220kV）
5. 静置结束后取油样送化验

> 参数来源：electrical_install.md §1.5~1.6

#### 6.5.3 关键技术参数

| 参数 | 500kV | 220kV | 依据 |
|------|-------|-------|------|
| 滤油机出口温度 | 60℃~70℃ | 60℃~70℃ | — |
| 油箱内最低温度 | ≥40℃ | ≥40℃ | — |
| 循环时间 | ≥48 h | ≥24 h | — |
| 循环油量 | ≥3 倍总油量 | ≥3 倍总油量 | — |
| 静置时间 | ≥72 h | ≥48 h | — |

> 参数来源：electrical_install.md §1.5~1.6

#### 6.5.4 质量控制要点

- 循环过程持续监控油温
- 静置期间禁止对变压器施加任何外力

#### 6.5.5 注意事项

- **跨章节一致性**：滤油机型号/数量 → Ch5.3 设备清单

### 3.5 整体密封试验（6.6）

#### 6.6.1 施工工艺流程

```
热油循环/静置完成 → 密封面检查 → 充压至 0.03 MPa（油箱顶部）
→ 保持 24h → 检查渗漏和变形 → 合格 → 进入试验阶段
```

#### 6.6.2 施工方法

1. 试验前全面检查所有密封面、法兰接口
2. 缓慢加压至油箱顶部 0.03 MPa
3. 保持压力 24h
4. 逐一检查所有密封点，记录观测结果

> 参数来源：electrical_install.md §1.7

#### 6.6.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 试验压力（油箱顶部） | 0.03 MPa | GB 50150 |
| 持续时间 | 24 h | — |
| 合格标准 | 无渗漏油，无永久变形 | — |

> 参数来源：electrical_install.md §1.7

#### 6.6.4 质量控制要点

- 加压过程缓慢、均匀，防止突然冲击
- 24h 内多次巡检，重点关注法兰和阀门

#### 6.6.5 注意事项

- 密封试验不合格的部位必须重新处理并复试
- **跨章节一致性**：试验记录 → Ch7 质量验收资料

### 3.6 建筑电气与照明（6.7）【如适用】

#### 6.7.1 施工工艺流程

```
安装准备 → 管路敷设/桥架安装 → 管内穿线/电缆敷设
→ 灯具/开关/插座安装 → 配电箱安装 → 通电试运行
```

#### 6.7.2 施工方法

1. 暗敷管路保护层厚度 ≥15mm，消防管路 ≥30mm
2. 弯曲半径：普通 ≥6 倍管外径，埋地/楼板 ≥10 倍管外径
3. 穿线前清理管内积水杂物，穿线后测绝缘电阻 ≥0.5 MΩ
4. 配电箱壁挂式底边距地 1.5m，垂直度偏差 ≤1.5‰
5. 插座接线：单相"左零右火"，三孔"左零右火上接地"，PE 线严禁串联
6. 相色规定：A 相黄、B 相绿、C 相红、N 线淡蓝、PE 线黄绿双色

> 参数来源：electrical_install.md §2.1~2.4

#### 6.7.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 暗敷保护层 | ≥15 mm | GB 50303 |
| 消防管保护层 | ≥30 mm | — |
| 弯曲半径（普通） | ≥6 倍管外径 | — |
| 弯扁度 | ≤管外径 10% | — |
| 绝缘电阻 | ≥0.5 MΩ | — |
| 配电箱底边距地 | 1.5 m | — |
| 同室高度差 | ≤5 mm | — |

> 参数来源：electrical_install.md §2.2~2.4

#### 6.7.4 质量控制要点

- 管路畅通、接头牢固、不漏浆
- 绝缘电阻测试记录完整
- 通电前逐回路校验

#### 6.7.5 注意事项

- **跨章节一致性**：电气材料（电线/管材/配电箱）→ Ch5.2 材料清单

### 3.7 防雷与接地（6.8）【如适用】

#### 6.8.1 施工工艺流程

```
接地体开挖/敷设 → 焊接搭接 → 防腐处理 → 回填
→ 设备接地连接 → 接地电阻测试 → 验收
```

#### 6.8.2 施工方法

1. 扁钢搭接长度为宽度的 2 倍，至少三面施焊
2. 圆钢搭接长度为直径的 6 倍，双面施焊
3. 焊接部位清除焊渣后涂刷沥青漆或防腐涂料
4. 主变接地点 2 点可靠接地
5. 铁芯接地电流 <100 mA
6. 金属管道、构架、电缆外皮均与接地干线等电位连接

> 参数来源：electrical_install.md §3.1~3.2

#### 6.8.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 扁钢搭接长度 | 宽度的 2 倍 | GB 50169 |
| 圆钢搭接长度 | 直径的 6 倍 | GB 50169 |
| 主变接地点 | 2 点 | — |
| 铁芯接地电流 | <100 mA | — |

> 参数来源：electrical_install.md §3.1~3.2

#### 6.8.4 质量控制要点

### 3.8 大型设备吊装（6.9）【如适用】

#### 6.9.1 施工工艺流程

**汽车起重机吊装**：

```
吊装方案审批 → 场地准备/基础承载力复核 → 起重机就位
→ 吊具检查/试吊（离地 200~300mm，悬停 10min）
→ 正式起吊 → 设备就位 → 吊索拆除
```

> 参数来源：special_general.md §4.5

#### 6.9.2 施工方法

1. 吊装前进行重量、重心、吊点、绳索强度计算
2. 正式吊装前进行试吊：离地 200~300mm，悬停 10min
3. 专人指挥，统一信号
4. 吊臂回转范围内设警戒区，非作业人员不得入内
5. 六级以上大风禁止高处吊装作业

> 参数来源：special_general.md §4.5

#### 6.9.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 试吊高度 | 离地 200~300 mm | — |
| 试吊悬停时间 | 10 min | — |
| 风力限制 | 六级以上大风禁止 | — |

#### 6.9.4 质量控制要点

- 吊装计算书审批完成
- 起重机年检合格、操作人员持证

#### 6.9.5 注意事项

- 大型设备吊装需编制**专项施工方案**并经审批
- **跨章节一致性**：起重机型号/吨位 → Ch5.3 设备清单；Ch8 须覆盖起重吊装危险源

## 6. 附录：完整输出示例

> 以下示例基于"500kV 电白变电站主变压器安装工程"场景，展示变电电气模板的理想输出。

```markdown
## 六、施工方法及工艺要求

### 6.1 施工工艺总流程

施工准备 → 本体就位检查 → 附件开箱检查及保管
→ 套管及套管 TA 试验 → 附件安装及器身检查（含冷却器、油枕、套管）
→ 抽真空及真空注油 → 热油循环 → 二次接线
→ 整体密封试验 → 变压器试验 → 结束

### 6.2 本体就位检查

1. 变压器到场后，检查运输密封完好性，取油样化验。
```
