## 3. 分项工程模板（6.2~6.7）

### 3.1 基础工程（6.2）

#### 6.2.1 施工工艺流程

**混凝土浇筑流程**：

```
基坑验收 → 垫层浇筑 → 钢筋绑扎 → 模板支设
→ 混凝土搅拌/运输 → 浇筑振捣 → 养护 → 拆模
```

**强夯地基处理流程**【如适用】：

```
场地平整 → 测量放线（标定夯点） → 第一遍点夯 → 推平夯坑
→ 第二遍点夯 → 推平夯坑 → 第一遍满夯 → 第二遍满夯
→ 整平碾压 → 检测验收
```

> 参数来源：civil_works.md §1.1 + §4.1

#### 6.2.2 施工方法

**混凝土浇筑**：

1. 浇筑前复核基础尺寸（顶面/底面对角根开、模板顶面高差、地螺规格及露长），合格后方可浇筑
2. 有地下水的基础须有合理排水措施，不得带水浇筑
3. 混凝土自由倾落高度不超过 2m，超过时使用串筒或溜槽
4. 分层浇筑，每层厚度不超过振动器作用深度（插式 ≤500mm，平板 ≤200mm）
5. 振动棒移动间距 ≤1.5R（R 为振动器作用半径），每点振捣 15~30s
6. 浇筑后 12h 内开始覆盖养护，普通混凝土 ≥7 天，掺缓凝剂/抗渗混凝土 ≥14 天

**强夯施工**【如适用】：

1. 试夯完成、形成施工参数后分段进行，从边缘向中央施夯
2. 起重机直线行驶，从一边向另一边推进
3. 每夯完一遍，推土机整平场地，按设计静置（等待超静孔隙水压力消散）
4. 最后一遍点夯完成后，以低能量进行满夯
5. 收锤标准：最后两击平均夯沉量不大于设计值（一般 50~100mm），夯坑周围无明显隆起

> 参数来源：civil_works.md §1.1, §4.2~4.4

#### 6.2.3 关键技术参数

**混凝土浇筑参数**：

| 自由倾落高度 | ≤2 m | GB 50666 |

#### 6.2.4 质量控制要点

- 混凝土坍落度现场检测每天不少于 3 次
- 同一铁塔四个基础埋深不等时，先施工深基础、后施工浅基础
- 强夯后检测应满足：压实系数 ≥0.94，承载力 ≥设计值
- 混凝土试块同条件养护和标准养护各留置足够组数

#### 6.2.5 注意事项

- **雨季措施**：浇筑前检查防滑，及时调整砂石含水率和坍落度；混凝土未凝时严禁雨水冲刷（详见 `special_general.md §1.2`）
- **冬季措施**：入模温度不低于 5℃，及时覆盖保温
- **跨章节一致性**：本节使用的设备（振动棒、串筒、强夯机等）→ Ch5.3 设备清单

### 3.2 钢筋工程（6.3）

#### 6.3.1 施工工艺流程

```
材料进场验收 → 钢筋加工（调直/切断/弯曲）→ 焊接/机械连接
→ 钢筋绑扎 → 保护层垫块安装 → 隐蔽工程验收
```

#### 6.3.2 施工方法

1. 钢筋进场后核对规格、数量，检查出厂合格证和试验报告
2. 钢筋加工前校对标高和保护层厚度
3. 接头采用直螺纹机械连接，接头等级 Ⅰ 级
4. 梁顶面主筋净间距 ≥30mm，利于混凝土灌筑
5. 保护层垫块间距 ≤1000mm，双层排列间距垫筋直径 25mm

> 参数来源：civil_works.md §2.1

#### 6.3.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 接头检验合格率 | ≥95% | GB 50204 |
| 每楼层/300 个接头抽检 | 1 组焊接接头 | GB 50204 |
| 梁顶面主筋净间距 | ≥30 mm | — |
| 保护层垫块间距 | ≤1000 mm | — |
| 搭接焊缝长度（单面焊） | ≥10d | — |
| 搭接焊缝长度（双面焊） | ≥5d | — |
| 焊缝宽度 | ≥0.7d | — |

> 参数来源：civil_works.md §2.1~2.2

#### 6.3.4 质量控制要点

- 电渣压力焊焊接面必须清除铁锈、油污
- 焊接接头取样送检，同批次同规格
- 钢筋保护层厚度实测偏差控制

#### 6.3.5 注意事项

- **雨季措施**：钢筋堆放场排水通畅，下垫木上盖塑料布；焊条受潮须烘干（碱性 300~350℃，酸性 100~150℃）（详见 `special_general.md §1.3`）
- **跨章节一致性**：钢筋规格/数量 → Ch5.2 材料清单

### 3.3 模板工程（6.4）

#### 6.4.1 施工工艺流程

```
模板设计/配制 → 涂刷脱模剂 → 模板安装定位 → 检查校正
→ 加固支撑 → 混凝土浇筑 → 养护达强度 → 拆模
```

#### 6.4.2 施工方法

1. 模板采用 15mm 酚醛木胶板，接缝处加 10mm 海绵条
2. 柱模板柱箍间距约 500mm，底部 1m 内加密，最底一道距底面 ≤300mm
3. 柱模安装后吊线检查垂直度，偏差 ≤3mm
4. 梁高 >500mm 时，梁侧模必须加设对拉螺栓
5. 梁跨度 >4m 时，按跨度的 1/1000~3/1000 起拱

> 参数来源：civil_works.md §3.1~3.3

#### 6.4.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 模板厚度 | 15 mm 酚醛木胶板 | — |
| 海绵条宽度 | 10 mm | — |
| 柱箍间距 | 约 500 mm | — |
| 底部加密最底一道 | 距底面 ≤300 mm | — |
| 柱模垂直度偏差 | ≤3 mm | GB 50204 |
| 梁侧模螺栓 | 梁高 >500mm 时必须加设 | — |
| 起拱值 | 跨度的 1/1000~3/1000 | — |

> 参数来源：civil_works.md §3.1~3.3

#### 6.4.4 质量控制要点

- 模板拼接严密，不漏浆
- 支撑系统稳固，检查后方可浇筑
- 拆模强度须满足规范要求

#### 6.4.5 注意事项

- **跨章节一致性**：模板材料 → Ch5.2 材料清单

### 3.4 砌体工程（6.5）【如适用】

#### 6.5.1 施工工艺流程

```
砌筑准备（弹线/排砖）→ 砌筑施工 → 构造柱/圈梁绑扎
→ 抹灰（底层→中层→面层）→ 护角处理 → 养护
```

#### 6.5.2 施工方法

1. 灰缝厚度控制在 8~12mm
2. 临时间断处做斜槎，长度 ≥高度的 2/3
3. 拉结筋沿墙高间距 ≤500mm，每边埋入长度 ≥500mm
4. 抹灰分三层：底层水泥砂浆约 5mm → 中层约 8~10mm → 面层混合砂浆约 5~7mm
5. 护角宽度每边 80mm，高度自地面起 2000mm

> 参数来源：civil_works.md §6.1~6.2

#### 6.5.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 灰缝厚度 | 8~12 mm | GB 50203 |
| 斜槎长度 | ≥高度的 2/3 | — |
| 拉结筋间距（沿墙高） | ≤500 mm | — |
| 拉结筋埋入长度（每边） | ≥500 mm | — |
| 底层抹灰厚度 | 约 5 mm | — |
| 中层抹灰厚度 | 约 8~10 mm | — |
| 面层抹灰厚度 | 约 5~7 mm | — |

> 参数来源：civil_works.md §6.1~6.2

#### 6.5.4 质量控制要点

- 砌体灰缝饱满度 ≥80%
- 墙体垂直度、平整度检测
- 抹灰表面无空鼓、裂缝

#### 6.5.5 注意事项

- **跨章节一致性**：砌体材料（砖、砂浆）→ Ch5.2 材料清单

### 3.5 屋面防水工程（6.6）【如适用】

#### 6.6.1 施工工艺流程

```
基层清理 → 找平层施工 → 防水层铺贴 → 保护层施工
→ 排水坡度调整 → 闭水试验 → 验收
```

#### 6.6.2 施工方法

1. 卷材防水搭接长度 ≥100mm
2. 铺贴方向从流水坡度下坡开始
3. 保护层采用细石混凝土，厚度 ≥40mm
4. 保护层分格缝间距 ≤6m

> 参数来源：civil_works.md §5.1

#### 6.6.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 卷材搭接长度 | ≥100 mm | GB 50207 |
| 保护层厚度（细石混凝土） | ≥40 mm | — |
| 保护层分格缝间距 | ≤6 m | — |
| 一般屋面坡度 | 1%~3% | — |
| 天沟/檐沟坡度 | ≥0.5% | — |

> 参数来源：civil_works.md §5.1~5.2

#### 6.6.4 质量控制要点

- 防水层与基层粘结牢固，无翘边、起泡
- 闭水试验 24h 无渗漏
- 排水坡度符合设计

#### 6.6.5 注意事项

- **雨季措施**：防水施工避开雨天，已铺卷材做好临时保护
- **跨章节一致性**：防水材料 → Ch5.2 材料清单

### 3.6 给排水安装（6.7）【如适用】

#### 6.7.1 施工工艺流程

```
管道预制/加工 → 管道安装 → 支架固定 → 管道试压
→ 卫生器具安装 → 消防管道安装 → 系统调试
```

#### 6.7.2 施工方法

1. 给水管安装完毕后进行水压试验，试验压力 0.6 MPa
2. 10min 内压力降 ≤0.05 MPa 为合格
3. 法兰螺栓外露长度 ≤2~3 丝

> 参数来源：civil_works.md §7.1

#### 6.7.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 给水管试验压力 | 0.6 MPa | GB 50242 |
| 10min 内压力降 | ≤0.05 MPa | — |
| 法兰螺栓外露 | ≤2~3 丝 | — |
| 洗脸盆离地高度 | 0.8 m | — |
| 消防喷头支管规格 | DN25 | — |

> 参数来源：civil_works.md §7.1~7.3

#### 6.7.4 质量控制要点

- 管道系统试压合格后方可隐蔽
- 阀门安装方向正确，开启灵活
- 消防系统联动测试

#### 6.7.5 注意事项

- **跨章节一致性**：管材型号 → Ch5.2 材料清单；水泵设备 → Ch5.3 设备清单

## 6. 附录：完整输出示例

> 以下示例基于"500kV 变电站主控通信楼基础及主体工程"场景，展示变电土建模板的理想输出。

```markdown
## 六、施工方法及工艺要求

### 6.1 施工工艺总流程

施工准备 → 基础工程 → 钢筋工程 → 模板工程
→ 混凝土浇筑与养护 → 砌体工程 → 屋面防水工程
→ 给排水安装 → 分项验收 → 竣工验收

### 6.2 基础工程

#### 6.2.1 施工工艺流程

基坑验收 → 垫层浇筑 → 钢筋绑扎 → 模板支设
→ 混凝土搅拌/运输 → 浇筑振捣 → 养护 → 拆模

#### 6.2.2 施工方法

1. 浇筑前复核基础顶面对角根开、底面对角根开、模板顶面高差、地螺规格及露长等，合格后方可浇筑。
2. 有地下水的基础须有合理有效的排水措施，不得带水浇筑。同一铁塔四个基础埋深不等时，先施工深基础，后施工浅基础。
3. 混凝土自由倾落高度不超过 2m，超过时必须使用串筒或溜槽。
4. 采用分层浇筑方式，每层厚度不超过 500mm（插式振动器），振动棒移动间距不超过 1.5 倍作用半径。
5. 每点振捣 15~30 秒，以表面泛浆、不再冒泡为准，做到不漏振、不过振。
6. 现场坍落度检测每天不少于 3 次，按《强制性条文》严格控制。
7. 浇筑后 12 小时内覆盖保湿膜并浇水养护，养护时间不少于 7 天。

#### 6.2.4 质量控制要点

- 坍落度现场检测每天不少于 3 次
- 混凝土试块按规范留置（标养 + 同条件）
- 拆模前强度须达到设计强度的 75%（侧模）或 100%（底模）

### 6.3 钢筋工程

#### 6.3.1 施工工艺流程

材料进场验收 → 钢筋调直/切断/弯曲 → 直螺纹机械连接
→ 钢筋绑扎 → 保护层垫块安装 → 隐蔽工程验收

#### 6.3.2 施工方法

1. 钢筋进场核对规格、数量，检查出厂合格证和力学试验报告。
2. 主筋接头采用直螺纹机械连接，接头等级 Ⅰ 级，拧紧力矩不小于 300 N·m。
3. 梁顶面主筋净间距不小于 30mm，利于混凝土灌筑。
4. 保护层垫块间距不超过 1000mm。

#### 6.3.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 接头检验合格率 | ≥95% | GB 50204 |
| 抽检频次 | 每 300 个接头 1 组 | GB 50204 |
| 主筋净间距 | ≥30 mm | — |
| 垫块间距 | ≤1000 mm | — |

### 6.4 模板工程

#### 6.4.1 施工工艺流程

模板配制 → 涂刷脱模剂 → 安装定位 → 检查校正 → 加固 → 浇筑 → 拆模

#### 6.4.2 施工方法

1. 采用 15mm 酚醛木胶板，接缝处加 10mm 海绵条。
2. 柱箍间距约 500mm，底部 1m 内加密。
3. 吊线检查垂直度，偏差不超过 3mm。
4. 梁高超过 500mm 时加设对拉螺栓，跨度超过 4m 时按 1/1000~3/1000 起拱。

### 6.5 砌体工程

1. 灰缝厚度 8~12mm，灰缝饱满度不低于 80%。
2. 拉结筋沿墙高间距不超过 500mm，每边埋入长度不小于 500mm。
3. 抹灰分三层施工：底层 5mm + 中层 8~10mm + 面层 5~7mm。
4. 阳角做护角处理，宽度每边 80mm，高度自地面起 2000mm。

### 6.6 屋面防水工程

1. 卷材搭接长度不小于 100mm，从流水坡度下坡开始铺贴。
2. 保护层采用细石混凝土，厚度不小于 40mm，分格缝间距不超过 6m。
3. 一般屋面坡度 1%~3%，天沟/檐沟坡度不小于 0.5%。

### 6.7 给排水安装

1. 给水管试验压力 0.6 MPa，10 分钟内压力降不超过 0.05 MPa。
2. 法兰螺栓外露 2~3 丝。
3. 消防喷头支管规格 DN25。
```
