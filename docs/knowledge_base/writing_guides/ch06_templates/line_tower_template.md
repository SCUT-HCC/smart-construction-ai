## 2. 施工工艺总流程（6.1）

**人工挖孔桩流程**：

```markdown
### 6.1 施工工艺总流程

测量分坑、场地清理 → 施工准备 → 孔口护壁制作
→ 孔口安全防护（软梯、安全绳、通风管线）→ 安装取土设备
→ 坑内气体检测 → 开挖土方 → 护壁制作 → 挖至设计桩深
→ 验孔 → 钢筋绑扎、支模 → 混凝土浇制及养护 → 拆模回填
```

> 参数来源：line_tower.md §1.1

**灌注桩流程**【如适用】：

```
测量定位 → 场地平整 → 护筒埋设 → 钻机就位
→ 钻孔（循环泥浆护壁）→ 清孔 → 终孔验收
→ 钢筋笼制作与安装 → 混凝土灌注 → 拔出护筒 → 养护
```

> 参数来源：line_tower.md §2.1

**钢结构屋面流程**【如适用】：

```
准备工作 → 楼承板（桁架）安装固定 → 纵向钢筋绑扎
→ 混凝土泵送浇筑 → 混凝土找平 → 混凝土养护
```

> 参数来源：line_tower.md §3.1

## 3. 分项工程模板（6.2~6.8）

### 3.1 测量放线（6.2）

#### 6.2.1 施工工艺流程

```
线路复测 → 桩位复核 → 分坑定位（直线塔/转角塔）
→ 基准桩设定 → 偏心基础校核（如有）
```

#### 6.2.2 施工方法

**直线塔基础分坑**：

1. 在塔位中心桩处安平经纬仪，前视辅助桩，水平度盘对零
2. 顺时针旋转 45°，按分坑尺寸表中 OB 数据定出基础中心
3. 以基础中心为圆心，以设计坑口直径 R/2 为半径划圆
4. 顺时针旋转望远镜至 135°、225°、315° 定出其余三个坑
5. 地形高差较大时，用斜距和角度换算平距：L₁ = L × cosα

**转角塔基础分坑**：

1. 先确定角平分线方向：水平向内旋转 (180°−θ)/2
2. 以角平分线为基准，按直线塔分坑方式进行四腿定位
3. 有预偏心基础时，必须按设计偏心数据分坑

> 参数来源：DOC 1 高密度片段 + line_tower.md §1

#### 6.2.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 分坑方法 | 45° 角度法 | — |
| 基准仪器 | 经纬仪 | — |
| 斜距换算 | L₁ = L × cosα | — |

**灌注桩预偏参数**【如适用】：

| 转角度数 α | 预偏值 Δh |
|-----------|----------|
| 0° < α ≤ 3° | 3B/1000 |
| 3° < α ≤ 20° | 4B/1000 |
| 20° < α ≤ 40° | 8B/1000 |
| 40° < α ≤ 60° | 10B/1000 |
| 60° < α ≤ 90° 及终端 | 12B/1000 |

> B = 基础根开值。参数来源：line_tower.md §2.3

#### 6.2.4 质量控制要点

- 复测中心桩偏离应在允许范围内，超限须报设计
- 交叉跨越物高程须实测复核
- 分坑定位误差按规范控制

#### 6.2.5 注意事项

- 丢失桩位须按原设计数据恢复
- **跨章节一致性**：测量仪器（经纬仪、钢尺）→ Ch5.3 设备清单

### 3.2 土方开挖与护壁（6.3）【人工挖孔桩适用】

#### 6.3.1 施工工艺流程

```
开挖前检查 → 分节开挖（每节 ≤1.0m）→ 护壁浇筑
→ 下节开挖 → 护壁浇筑 → ... → 挖至设计桩深 → 验孔
```

#### 6.3.2 施工方法

1. 每节开挖深度不超过 1.0m
2. 第一节护壁高出地面 100mm，设锁口
3. 护壁混凝土强度 ≥C25
4. 上下节搭接长度 ≥50mm，竖向钢筋有效搭接
5. 尽可能采用风镐机械开挖，人工修整成型
6. 成孔后项目部和监理双方代表验孔，合格后立即安装钢筋笼、浇筑混凝土
7. 土方由专人提运至指定堆土场，不得堆放在孔口 5m 范围内

> 参数来源：line_tower.md §1.2

#### 6.3.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 每节开挖深度 | ≤1.0 m | — |
| 第一节护壁高出地面 | 100 mm | — |
| 护壁混凝土强度 | ≥C25 | — |
| 上下节搭接长度 | ≥50 mm | — |
| 余土堆放禁区 | 孔口 5m 范围内 | — |

> 参数来源：line_tower.md §1.2

#### 6.3.4 质量控制要点

- 护壁厚度均匀，无蜂窝
- 孔径偏差控制
- 孔底沉渣厚度符合设计

#### 6.3.5 注意事项

- 山地基坑尽量不降基或少降基，采取开挖小平台方式
- **雨季措施**：防止降雨软化坑壁土质，成孔后应及时浇筑
- **跨章节一致性**：风镐 → Ch5.3 设备清单；Ch8 须覆盖基坑坍塌危险源

### 3.3 通风与气体检测（6.4）【人工挖孔桩适用】

#### 6.4.1 施工工艺流程

```
每日开工前气体检测 → 合格 → 下人作业
→ 每 2 小时定时检测 → 异常 → 停止作业/通风 → 复测合格 → 恢复
```

#### 6.4.2 施工方法

1. 每日开工前检测，作业中每 2 小时检测一次
2. 孔深 >10m 或检出有害气体时必须强制通风
3. 送风时间 ≥5min
4. 使用毒气检测仪，检测氧浓度、可燃气体和一氧化碳
5. 检测不合格时禁止下人作业

> 参数来源：line_tower.md §1.3

#### 6.4.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 检测频率 | 开工前 + 每 2h | — |
| 强制通风阈值 | 孔深 >10m 或有害气体 | — |
| 送风时间 | ≥5 min | — |
| 氧浓度安全范围 | 19.5%~23.5% | GBZ 2 |
| 可燃气体浓度 | <最低爆炸极限 10% | — |
| CO 浓度 | <25 ppm | — |

> 参数来源：line_tower.md §1.3 + special_general.md §2.3

#### 6.4.4 质量控制要点

- 检测记录完整（时间、地点、气体种类、浓度）
- 通风设备运行正常

#### 6.4.5 注意事项

- 此为人工挖孔桩**安全生命线**，不得省略
- **跨章节一致性**：鼓风机、毒气检测仪 → Ch5.3；Ch8 须覆盖孔内缺氧/中毒危险源

### 3.4 钢筋笼制安（6.5）

#### 6.5.1 施工工艺流程

```
钢筋加工（调直/切断/弯曲）→ 钢筋笼绑扎/焊接
→ 分段制作（如过长）→ 吊装入孔 → 孔内焊接搭接
→ 定位校核 → 准备浇筑
```

#### 6.5.2 施工方法

1. 钢筋笼过长时分段制作，入孔后焊接
2. 临时加劲架间距 3~4m，安放后拆除
3. 按规定吊点位置绑扎，使用起重设备吊装
4. 孔底未清理干净时严禁强行下压钢筋笼
5. 主筋接头采用直螺纹机械连接

> 参数来源：line_tower.md §1.4

#### 6.5.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 临时加劲架间距 | 3~4 m | — |
| 接头方式 | 直螺纹机械连接 | — |
| 接头等级 | Ⅰ级 | — |

**直螺纹接头安装要求**：
- 钢筋规端面垂直于轴线
- 丝头加工长度满足要求
- 拧紧力矩不小于规定值

**锥螺纹接头**【如适用】：
- 连接件规格与钢筋匹配
- 拧紧力矩符合产品说明

> 参数来源：line_tower.md §1.4 + DOC 1 高密度片段

#### 6.5.4 质量控制要点

- 钢筋笼主筋间距偏差控制
- 接头质量取样送检
- 保护层垫块牢固

#### 6.5.5 注意事项

- **跨章节一致性**：钢筋规格/数量 → Ch5.2 材料清单

### 3.5 混凝土浇筑（6.6）

#### 6.6.1 施工工艺流程

**干式浇筑**（孔底积水 ≤50mm）：

```
孔底清理 → 串筒安装（落距 >3m 时）→ 混凝土浇筑
→ 振捣 → 分层浇筑至设计标高 → 养护
```

**水下浇筑**【积水较多且无法抽干时适用】：

```
导管安装（φ250~300mm）→ 首批混凝土灌注（满足初始埋管深度）
→ 连续灌注 → 导管始终埋入混凝土面以下 → 灌注至设计标高
```

#### 6.6.2 施工方法

**干式浇筑**：

1. 落距 >3m 时必须使用串筒，末端距孔底 ≤2m
2. 优先采用商品混凝土，运输困难时现场搅拌
3. 商品混凝土出仓 2 小时内浇筑完成
4. 砂、石、水泥、添加剂须经第三方检测合格

**水下浇筑**：

1. 采用导管法（水下混凝土灌注法）
2. 导管直径 250~300mm
3. 首批灌注量须满足导管初次埋置深度要求
4. 灌注过程中导管不得提出混凝土面

> 参数来源：line_tower.md §1.5

**灌注桩参数**【如适用】：

| 桩身混凝土强度 | C35 | 设计文件 |
| 承台垫层强度 | C20 | — |
| 水胶比 | 0.55 | — |
| 坍落度 | 180~220 mm | — |

> 参数来源：line_tower.md §2.2

| 参数 | 标准值 | 依据 |

> 参数来源：line_tower.md §1.5

#### 6.6.4 质量控制要点

- 坍落度现场检测
- 混凝土试块留置
- 灌注桩充盈系数 ≥1.0

### 3.6 灌注桩钻孔工艺（6.7）【灌注桩适用】

#### 6.7.1 施工工艺流程

```
护筒埋设 → 钻机就位/调平 → 钻孔（循环泥浆护壁）
→ 一次清孔 → 终孔验收（孔深/孔径/垂直度）
→ 下钢筋笼 → 安装导管 → 二次清孔 → 混凝土灌注
```

#### 6.7.2 施工方法

1. 护筒中心与桩位偏差 ≤50mm
2. 钻孔过程中泥浆循环护壁，控制泥浆比重
3. 一次清孔后检查孔底沉渣
4. 终孔验收：孔深、孔径、垂直度偏差
5. 钢筋笼下放后二次清孔
6. 混凝土灌注采用导管法

> 参数来源：line_tower.md §2.1~2.2

#### 6.7.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 护筒偏差 | ≤50 mm | — |
| 沉渣厚度（摩擦桩） | ≤200 mm | JGJ 94 |
| 沉渣厚度（端承桩） | ≤50 mm | JGJ 94 |

**导线对地最小距离（500kV 线路）**【如适用】：

| 地区类型 | 最小距离(m) | 导线状态 |
|---------|-----------|---------|
| 居民区 | 14 | 40℃弧垂 |
| 非居民区 | 11 | 40℃弧垂 |
| 交通困难地区 | 8.5 | 40℃弧垂 |

> 参数来源：line_tower.md §2.4

#### 6.7.4 质量控制要点

- 钻孔垂直度偏差 ≤1%
- 桩基检测：声波透射法/低应变动测

#### 6.7.5 注意事项

- **跨章节一致性**：钻机型号 → Ch5.3；泥浆材料 → Ch5.2

### 3.7 钢结构屋面（6.8）【GIS 室楼承板适用】

#### 6.8.1 施工工艺流程

```
准备工作 → 楼承板安装固定 → 栓钉（剪力钉）焊接
→ 纵向钢筋绑扎 → 混凝土泵送浇筑 → 找平 → 养护
```

#### 6.8.2 施工方法

1. 楼承板在钢梁上支承长度 ≥50mm
2. 侧向连接采用咬口钳，端部熔焊连接
3. 焊点直径 ≥20mm，间距 ≤300mm
4. 栓钉焊接前清除钢梁表面油漆和铁锈
5. 开孔 <500×500mm 的浇筑后切割无需加强
6. 开孔 >500mm 按设计加设洞口边梁
7. 混凝土浇筑方向垂直于压型钢板波纹方向
8. 振捣工具不得直接接触压型钢板
9. 初凝前二次抹压，防止表面裂缝
10. 养护时间 ≥7 天

> 参数来源：line_tower.md §3.2~3.4

#### 6.8.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 楼承板支承长度 | ≥50 mm | — |
| 焊点直径 | ≥20 mm | — |
| 焊点间距 | ≤300 mm | — |
| 栓钉弯曲试验 | 30° 焊缝无裂纹 | — |
| 浇筑方向 | 垂直于波纹方向 | — |
| 养护时间 | ≥7 天 | — |

> 参数来源：line_tower.md §3.2~3.4

#### 6.8.4 质量控制要点

- 栓钉焊接质量抽检（弯曲试验）
- 混凝土保护层厚度
- 楼承板接缝密封处理

#### 6.8.5 注意事项

- 屋脊位置接缝处须密封胶防水处理，缝隙 >20mm 时用同等镀锌铁皮填充
- **跨章节一致性**：栓钉焊机 → Ch5.3；楼承板/钢筋 → Ch5.2

## 6. 附录：完整输出示例

> 以下示例基于"500kV 输电线路人工挖孔桩基础施工"场景，展示线路塔基模板的理想输出。

```markdown
## 六、施工方法及工艺要求

### 6.1 施工工艺总流程

测量分坑、场地清理 → 施工准备 → 孔口护壁制作
→ 孔口安全防护 → 安装取土设备 → 坑内气体检测
→ 开挖土方 → 护壁制作 → 挖至设计桩深
→ 验孔 → 钢筋绑扎、支模 → 混凝土浇制及养护 → 拆模回填

### 6.2 测量放线

#### 直线塔基础分坑

采用 45° 角度法分坑。在塔位中心桩处安平经纬仪，前视辅助桩，将水平度盘对零，顺时针旋转 45°，按分坑尺寸表中 OB 数据定出基础中心，再以基础中心为圆心、设计坑口直径 R 一半为半径划圆。再顺时针旋转望远镜至 135°、225°、315° 可定出其余三个坑。

如遇地形高差较大，用斜距和角度换算成平距：L₁ = L × cosα。

#### 转角塔基础分坑

转角塔分坑先定出角平分线方向，即水平向内旋转 (180°−θ)/2，定出辅助桩，再以角平分线为基准按直线塔方式进行四腿定位。有预偏心基础时，必须按设计偏心数据分坑。

### 6.3 土方开挖与护壁

1. 根据土质情况采用机械开挖、人工修整成型，配备机械化吊土设备和鼓风送氧设备。
2. 每节开挖深度不超过 1.0m，第一节护壁高出地面 100mm 设锁口。
3. 护壁混凝土强度不低于 C25，上下节搭接长度不小于 50mm。
4. 坑口固定"井"字起吊架，吊土时洞底人员不得站在吊桶正下方。
5. 余土不得堆放在洞口 5m 范围内，山地基坑应选取偏离基坑且靠平缓区域临时堆放。

### 6.4 通风与气体检测

每日开工前及作业中每 2 小时进行坑内气体检测。孔深超过 10m 或检出有害气体时必须强制通风，送风时间不少于 5 分钟。

| 检测指标 | 安全范围 |
|---------|---------|
| 氧浓度 | 19.5%~23.5% |
| 可燃气体 | <最低爆炸极限 10% |
| 一氧化碳 | <25 ppm |

### 6.5 钢筋笼制安

1. 主筋接头采用直螺纹机械连接。
2. 钢筋笼过长时分段制作，入孔后焊接搭接，临时加劲架间距 3~4m。
3. 孔底未清理干净时严禁强行下压钢筋笼。

### 6.6 混凝土浇筑

优先采用商品混凝土，运输困难时现场搅拌。混凝土出仓 2 小时内浇筑完成。落距超过 3m 时使用串筒，末端距孔底不超过 2m。

基坑验收合格后应立即安装钢筋笼并浇筑混凝土，防止降雨软化坑壁土质。

### 6.7 验收标准

| 检验项目 | 合格标准 | 依据 |
|---------|---------|------|
| 桩位偏差 | ≤设计允许值 | GB 50026 |
| 护壁混凝土 | ≥C25 | — |
| 钢筋接头合格率 | ≥95% | GB 50204 |
| 混凝土强度 | 符合设计等级 | GB 50204 |
```
