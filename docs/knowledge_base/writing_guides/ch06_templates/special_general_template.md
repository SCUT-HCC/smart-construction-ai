## 1. 模板使用说明

### 1.1 适用判断

当 `ch06_施工方法.md` Step 1 决策树输出为**特殊作业**或**通用**时，加载本模板。
关键词匹配：含"防腐/防火涂料/涂装/雨季/有限空间/起重/吊装"。

### 1.2 组合式模板说明

本模板为**菜单式结构**，Agent 根据具体专项工程选择需要的小节组合，非全部适用。

**典型组合**：

| 专项工程 | 选用小节 |
|---------|---------|
| 构支架防腐涂装 | 6.1 + 6.2 + 6.N |
| 钢结构防火涂料 | 6.1 + 6.3 + 6.N |
| 防腐 + 防火综合涂装 | 6.1 + 6.2 + 6.3 + 6.N |
| 雨季施工专项方案 | 6.1 + 6.4 + 6.N |
| 有限空间作业方案 | 6.1 + 6.5 + 6.N |
| 起重吊装专项方案 | 6.1 + 6.6 + 6.N |
| 绿色施工专项措施 | 6.1 + 6.7 + 6.N |

### 1.3 调用步骤

1. 从 Ch2（工程概况）获取专项工程类型
2. 确认需要的小节组合 → 裁剪不适用小节
3. 从 `process_references/special_general.md` 对应章节提取参数
4. 从 `fragments.jsonl` 检索涂装工程/专题方案类片段参考措辞
5. 完成后执行 §5 跨章节一致性检查

## 3. 分项工程模板（6.2~6.7）

### 3.1 防腐涂装（6.2）【如适用】

#### 6.2.1 施工工艺流程

```
表面检查 → 喷砂/抛丸除锈（Sa2.5 级）→ 4h 内涂刷底漆
→ 底漆干燥 → 涂刷中间漆 → 中间漆干燥
→ 涂刷面漆 → 面漆干燥 → 涂层检测 → 验收
```

#### 6.2.2 施工方法

1. 除锈等级达到 Sa2.5（喷砂/抛丸），表面无可见油脂、污垢、氧化皮、铁锈和旧涂层
2. 除锈后 4 小时内涂刷底漆
3. 禁止作业条件：相对湿度 >85% 或钢材表面温度低于露点 3℃
4. 施工方式：高压无气喷涂
5. 喷枪与工件距离 250~350mm，角度垂直于工件表面
6. 移动速度均匀，上一道干燥后方可涂下一道
7. 涂层质量要求：平整光滑，色泽一致，无流挂、起泡、剥落

> 参数来源：special_general.md §3.1~3.2

#### 6.2.3 关键技术参数

**涂层体系**：

| 涂层 | 涂料类型 | 干膜厚度 | 依据 |
|------|---------|---------|------|
| 底漆 | 环氧富锌底漆 | ≥50 μm | 设计文件 |
| 中间漆 | 环氧云铁中间漆 | ≥100 μm | 设计文件 |
| 面漆 | 聚氨酯面漆或氟碳面漆 | ≥50 μm | 设计文件 |
| **总厚度** | — | **≥200 μm** | — |

**涂装操作参数**：

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 除锈等级 | Sa2.5 | GB/T 8923 |
| 底漆涂装时限 | 除锈后 4h 内 | — |
| 湿度限制 | <85% | — |
| 喷枪距离 | 250~350 mm | — |
| 喷枪角度 | 垂直于工件 | — |

> 参数来源：special_general.md §3.1~3.2

#### 6.2.4 质量控制要点

- 除锈等级现场比对样板确认
- 每道涂层干膜厚度实测
- 附着力测试（划格法/拉拔法）

#### 6.2.5 注意事项

- 涂装作业严禁在大风（六级以上）或雨雪天气进行
- **跨章节一致性**：涂料品牌/型号 → Ch5.2 材料清单；喷砂机/喷枪 → Ch5.3 设备清单

### 3.2 防火涂料（6.3）【如适用】

#### 6.3.1 施工工艺流程

```
基层防锈验收 → 界面处理 → 第一遍喷涂（盖住基层为准）
→ 表干（4~24h）→ 第二遍喷涂 → ... → 达到设计耐火极限厚度
→ 质量检测 → 验收
```

#### 6.3.2 施工方法

1. 基层处理：防锈底漆涂装完毕并验收合格后方可施工
2. 第一遍以盖住基层为准，防止流坠
3. 薄型防火涂料每层喷涂厚度 ≤3mm
4. 厚型防火涂料每层喷涂厚度 5~8mm
5. 层间间隔：前遍表干后（4~24 小时）
6. 逐层施工直至达到设计耐火极限要求的厚度

> 参数来源：special_general.md §3.3

#### 6.3.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 薄型每层厚度 | ≤3 mm | GB 14907 |
| 厚型每层厚度 | 5~8 mm | GB 14907 |
| 层间间隔 | 4~24 h（前遍表干后） | — |
| 裂纹宽度限制（厚型） | ≤1 mm | — |
| 合格标准 | 无空鼓、无脱落 | — |
| 厚度要求 | 满足设计耐火极限 | 设计文件 |

> 参数来源：special_general.md §3.3

#### 6.3.4 质量控制要点

- 涂层厚度实测（测厚仪）
- 外观检查：无空鼓、无脱落、裂纹宽度 ≤1mm
- 留置同条件试板

#### 6.3.5 注意事项

- 防火涂料施工前须确认产品检验报告和型式认可证书
- **跨章节一致性**：防火涂料型号 → Ch5.2；喷涂设备 → Ch5.3

### 3.3 雨季施工措施（6.4）【如适用】

#### 6.4.1 总体原则

| 原则 | 要求 |
|------|------|
| 防雨 | 小雨不停工，大雨大风停止室外施工 |
| 排水 | 施工场地做好排水沟规划 |
| 防护 | 露天机械搭设防晒防雨防砸安全棚 |
| 持证 | 特殊工种操作人员必须持证上岗 |

#### 6.4.2 施工方法

**混凝土工程雨季措施**：

1. 浇筑前检查脚手架牢固性和防滑措施，掌握天气情况
2. 及时测定砂石含水率，调整配合比，严格控制坍落度
3. 混凝土未凝时严禁雨水冲刷，现场必须有足够覆盖材料
4. 合理预留施工缝，调整工作时间

**钢筋及焊接雨季措施**：

1. 钢筋堆放场地平整排水通畅，下垫木，雨前覆盖塑料布
2. 雨天无防雨措施时禁止焊接，焊后接头不能立即接触雨水
3. 焊条受潮须烘干：碱性低氢型 300~350℃，酸性型 100~150℃
4. 烘焙后存于 100℃ 恒温箱，随用随取

**临时用电防护**：

1. 配电箱上锁，有防雨盖板和危险标志
2. 禁止使用破损或绝缘不良电线，禁止电线随地走
3. 所有机具装有漏电保护装置，执行"一机、一闸、一漏、一箱"
4. 基坑内外必须配备足够照明灯，设围栏和安全标识

**防台防风措施**：

1. 塔吊：吊钩升至最高限位以下，小车收至臂根，大臂顺风自由回转
2. 脚手架：检查并加固连墙件
3. 切断非必要用电设备电源

> 参数来源：special_general.md §1.1~1.5

#### 6.4.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 焊条烘干温度（碱性） | 300~350℃ | — |
| 焊条烘干温度（酸性） | 100~150℃ | — |
| 焊条保温箱温度 | 100℃ | — |
| 漏电保护制度 | 一机一闸一漏一箱 | JGJ 46 |

> 参数来源：special_general.md §1.3~1.4

#### 6.4.4 质量控制要点

- 雨后复工前检查排水设施和机械设备
- 混凝土坍落度调整记录

#### 6.4.5 注意事项

- 雨季措施可作为独立章节，也可嵌入其他工程类型模板的"注意事项"段
- **跨章节一致性**：Ch8 须覆盖雨季触电、滑倒等危险源

### 3.4 有限空间作业（6.5）【如适用】

#### 6.5.1 施工工艺流程

```
作业审批 → 风险辨识 → 通风处理 → 气体检测合格
→ 佩戴防护用品 → 设置监护 → 作业 → 定时复测 → 撤离
```

#### 6.5.2 施工方法

**七不准原则**：

1. 未经风险辨识不准作业
2. 未经通风和检测合格不准作业
3. 不佩戴劳动防护用品不准作业
4. 没有监护不准作业
5. 电气设备不符合规定不准作业
6. 未经审批不准作业
7. 未经培训演练不准作业

**总原则**：先通风 → 再检测 → 后作业

**通风要求**：

1. 横断面平均风速 ≥0.8 m/s 或换气次数 ≥20 次/h
2. 单出入口：送风口置于作业区域
3. 多出入口：近处送风，远处排风
4. 死角处理：设挡板或改变吹风方向

**监护要求**：

1. 专职监护人员保持与作业人员联系，不得离岗
2. 备齐应急装备：三脚架、安全绳、正压式空气呼吸器、防爆对讲机、急救箱
3. 检测仪报警或人员不适时立即停止作业并撤离

> 参数来源：special_general.md §2.1~2.4

#### 6.5.3 关键技术参数

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 横断面风速 | ≥0.8 m/s | GBZ 2 |
| 换气次数 | ≥20 次/h | — |
| 氧浓度安全范围 | 19.5%~23.5% | GBZ 2 |
| 可燃气体浓度 | <最低爆炸极限 10% | — |
| CO 浓度 | <25 ppm | — |
| 检测点数（高度 ≤8m） | ≥2 个 | — |
| 上检测点 | 距顶部 ≤1 m | — |
| 下检测点 | 距底部 ≤1 m | — |

> 参数来源：special_general.md §2.2~2.3

#### 6.5.4 质量控制要点

- 通风和检测记录完整（时间、地点、气体种类、浓度）
- 作业审批手续齐全
- 应急演练已完成

#### 6.5.5 注意事项

- 有限空间作业属**高危作业**，需编制专项方案
- **跨章节一致性**：气体检测仪/通风设备/应急装备 → Ch5.3；Ch8 须覆盖中毒窒息危险源；Ch9 须有中毒窒息应急处置

### 3.5 起重吊装（6.6）【如适用】

#### 6.6.1 施工工艺流程

**桥式起重机安装**：

```
基础验收 → 组装轨道/行走机构 → 安装主梁/端梁
→ 安装电气系统 → 安装起升机构 → 试运转 → 验收
```

**塔式起重机安装**：

```
选定位置 → 浇筑承台基础（预埋螺栓）→ 安装基础节
→ 安装顶升套架 → 安装过渡节 → 安装回旋总成
→ 安装平衡臂 → 安装起重臂 → 安装配重
→ 电气接线、安装钢丝绳 → 试运转 → 顶升塔身 → 整机调试
```

**塔吊拆除**：

```
拆卸平衡重（保留一块）→ 拆卸起重臂 → 拆卸剩余平衡重
→ 拆卸平衡臂 → 拆卸驾驶室及回转总成
→ 拆卸标准节及附墙装置 → 拆卸底架及基础节
```

> 参数来源：special_general.md §4.1~4.3

#### 6.6.2 施工方法

**桥式起重机**：

1. 安装前复查跨度、跨度差、对角线差、主梁上拱度、主梁旁弯度、车轮偏斜
2. 空负荷升降操作不少于三次
3. 逐渐加负荷做起升运转直至额定负荷
4. 静负荷试验：额定 125% 载荷悬吊 10min
5. 动负荷试验：额定 110% 载荷全行程往返运行

**通用安全要点**：

1. 吊装前进行重量、重心、吊点、绳索强度计算
2. 正式吊装前试吊：离地 200~300mm，悬停 10min
3. 专人指挥，统一信号
4. 吊臂回转范围内设警戒区
5. 六级以上大风禁止高处吊装作业
6. 顶升过程中禁止回转、起升、变幅；四级以上大风禁止顶升

> 参数来源：special_general.md §4.1~4.5

#### 6.6.3 关键技术参数

**桥式起重机安装精度**：

| 检测项目 | 质量指标 | 检测方法 |
|---------|---------|---------|
| 跨度偏差 | ≤允许值 | 钢尺量 |
| 对角线差 | ≤允许值 | 钢尺量 |
| 主梁上拱度 | 符合设计 | 拉线量 |
| 车轮水平偏斜 | ≤L/1000 | 经纬仪 |

**通用安全参数**：

| 参数 | 标准值 | 依据 |
|------|-------|------|
| 试吊高度 | 离地 200~300 mm | — |
| 试吊悬停 | 10 min | — |
| 静负荷试验 | 额定 125% | GB 50278 |
| 动负荷试验 | 额定 110% | GB 50278 |
| 高处吊装风力限制 | 六级以上禁止 | — |
| 顶升风力限制 | 四级以上禁止 | — |

> 参数来源：special_general.md §4.4~4.5

#### 6.6.4 质量控制要点

- 吊装计算书经审批
- 起重机年检合格证有效
- 操作人员持证上岗
- 安全装置（限位器、力矩限制器等）功能正常

#### 6.6.5 注意事项

- 起重吊装属**危险性较大的分部分项工程**，需编制专项方案并经专家论证
- **跨章节一致性**：起重机型号/吨位 → Ch5.3；吊具/钢丝绳 → Ch5.2；Ch8 须覆盖起重伤害危险源

### 3.6 绿色施工措施（6.7）【如适用】

#### 6.7.1 扬尘控制（"六个 100%"）

| 措施 | 要求 |
|------|------|
| 围挡 | 连续封闭，市区 ≥2.5m，一般 ≥1.8m |
| 裸土覆盖 | 超过 48h 不施工的裸土必须覆盖 |
| 车辆冲洗 | 出入口设冲洗设施，净车出场 |
| 洒水降尘 | 土方作业必须洒水喷雾 |
| 道路硬化 | 施工主要道路硬底化处理 |
| 密闭运输 | 运输车辆密闭或覆盖 |

#### 6.7.2 噪音与光污染

| 项目 | 要求 |
|------|------|
| 夜间施工禁令 | 22:00~次日 6:00 禁止噪声施工 |
| 特殊作业 | 办理许可证并公告居民 |
| 设备 | 选用低噪声设备，搭建防噪棚 |
| 电焊 | 遮挡弧光 |

#### 6.7.3 水资源与排放

| 项目 | 要求 |
|------|------|
| 循环用水 | 设循环水系统 |
| 生活污水 | 厕所设化粪池，食堂设隔油池 |
| 生产废水 | 经沉淀池达标后排入市政管网 |

> 参数来源：special_general.md §5~6

#### 6.7.4 质量控制要点

- 扬尘监测记录
- 噪音监测记录
- 废水排放检测合格

#### 6.7.5 注意事项

- 绿色施工内容可独立成章（Ch10），也可嵌入 Ch6 作为通用措施小节
- 与 Ch8.6（环境保护）内容区分：本节侧重施工方法层面的绿色措施

### 6.2 表面处理
