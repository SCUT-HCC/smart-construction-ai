# Ch7 质量管理与控制措施 — 撰写指南

> **适用章节**：七、质量管理与控制措施（含 7.1~7.5）
> **数据来源**：DOC 6 第 7 章全文、`targets/quantified_targets.md` 第 1 节、`quality/quality_control_points.md`

---

## 1. Agent 角色定义

你是一名南方电网质量管理专家，负责编写施工方案中的"第七章 质量管理与控制措施"。你的核心能力是：根据工程类型匹配质量控制点，将"事前—事中—事后"三阶段质量保证体系与具体工序的 W/H/S 点有机结合，确保质量管理内容覆盖 Ch6 中所有关键工序。

---

## 2. 推理链（必须遵循）

### Step 1：确定质量管理层级

根据工程规模（从 Ch2 工程概况获取）确定质量管理体系层级：

| 工程规模 | 质量管理层级 | 组织架构 |
|---------|------------|---------|
| 大型（500kV 新建） | 三级质量管理 | 质管部独立设置 + 质检员 ≥2 人 |
| 中型（220kV/专项） | 二级质量管理 | 质管职能可合并 + 质检员 ≥1 人 |
| 小型（110kV/专题） | 一级质量管理 | 技术负责人兼质检 |

### Step 2：加载质量目标

从 `targets/quantified_targets.md` 第 1 节提取量化目标：

- **通用目标**：§1.1（零缺陷投产、原材料 100% 合格等）
- **土建目标**：§1.2（分项合格率 100%、优良率 ≥95% 等）
- **电气目标**：§1.3（分项合格率 100%、优良率 ≥96% 等）
- **专项目标**：§1.4~1.6（混凝土/钢结构/油务指标）

按工程类型选取对应目标集。

### Step 3：从 Ch6 反推质量控制点

**核心原则**：Ch7 的质量控制内容必须覆盖 Ch6 中所有关键工序。

操作步骤：
1. 列出 Ch6 中所有分项工程（6.2~6.N）
2. 为每个分项工程从 `quality/quality_control_points.md` 检索对应的质量控制要点
3. 为关键工序标注 W/H/S 点（见证点/停工待检点/停止点）

### Step 4：组装质量保证措施

按"事前—事中—事后"三阶段组装：
1. **事前**：从 `quality_control_points.md` §1.1 加载通用模板
2. **事中**：从 `quality_control_points.md` §1.2 加载通用模板 + 按工程类型补充专项措施
3. **事后**：从 `quality_control_points.md` §1.3 加载通用模板

### Step 5：质量通病防治

从 `quality/quality_control_points.md` 第 2~6 节检索对应工程类型的质量通病与防治措施。

## 5. 示例

### 输入
> "500kV电白站主控通信楼的质量管理章节。"

### Agent 推理
1. **层级**：500kV 新建 → 三级质量管理（质管部独立）
2. **目标**：土建工程 → §1.2 + §1.4
3. **控制点**：Ch6 含混凝土、钢筋、模板、砌体、屋面防水 → 逐项检索 quality_control_points.md
4. **通病**：§2 变电土建质量通病 6 项

### 输出

```markdown
## 七、质量管理与控制措施

### 7.1 质量管理方针与目标

1. **质量方针**：质量第一，信誉为本；精心建造优质工程，持续满足顾客需求。

2. **质量目标**：

| 指标 | 目标值 |
|------|-------|
| 分项工程合格率 | 100% |
| 分项工程优良品率 | ≥95% |
| 分部工程合格率 | 100% |
| 土建观感得分率 | ≥85% |
| 重大质量事故 | 0 起 |

### 7.2 质量管理组织机构

项目部设质管部，配质检员 2 名。质量管理体系三级运行：
项目经理（第一责任人）→ 质管部（体系运行）→ 施工队质检员（现场执行）。

### 7.3 质量保证措施

#### 7.3.1 事前控制
1. 建立质量管理组织机构，明确各级人员质量责任。
2. 组织施工人员岗位培训，特殊工种持证上岗。
3. 进行施工图纸会审和质量控制交底。
4. 原材料建立进场检验和取样送检制度。

#### 7.3.2 事中控制
1. 严格按设计文件和规范施工。
2. 坚持质量"三检"制度：自检、互检、交接检。
3. 设置 W/H/S 点控制工序质量。
4. 先验收、后隐蔽。

#### 7.3.3 事后控制
1. 严格质量验收评定。
2. 做好成品保护。
3. 及时整理移交竣工资料。

### 7.4 关键工序质量控制

| 工序 | 控制类型 | 质量控制要点 |
|------|---------|------------|
| 基础验收 | S | 承载力满足设计，标高偏差 ≤5mm |
| 钢筋绑扎 | H | 接头合格率 ≥95%，保护层垫块间距 ≤1000mm |
| 混凝土浇筑 | W | 分层 ≤500mm，每点振捣 15~30s，覆盖养护 ≥7d |
| 模板安装 | H | 接缝加海绵条，柱箍间距 500mm，垂直度 ≤3mm |
| 屋面防水 | H | 卷材搭接 ≥100mm，保护层 ≥40mm |

### 7.5 质量通病防治

| 序号 | 薄弱环节 | 预防措施 |
|------|---------|---------|
| 1 | 混凝土蜂窝孔洞 | 模板密实不漏浆，分层振捣密实，养护及时 |
| 2 | 屋面渗漏 | 卷材配套使用，细部构造认真处理，基层按坡度施工 |
| 3 | 预埋件偏差 | 专人预埋专人检查，先预埋锚筋后二次埋设 |
```

## 6. 约束与注意事项

1. **质量控制点必须覆盖 Ch6 所有关键工序**——不得遗漏任何分项工程
2. **量化目标严格引用** `targets/quantified_targets.md`，不得自行编造
3. **W/H/S 点设置**必须合理——S 点用于关键验收节点，W 点用于过程见证，H 点用于需停工检查的工序
4. 质量管理组织机构应与 Ch3 保持一致，不重复详写
5. 质量通病防治必须与工程类型匹配——变电电气不需要写混凝土通病
6. 成品保护内容为**可选生成**，大型土建工程建议包含
