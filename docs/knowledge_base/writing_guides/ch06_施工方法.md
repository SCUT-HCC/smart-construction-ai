# Ch6 施工方法及工艺要求 — 撰写指南

> **适用章节**：六、施工方法及工艺要求（含 6.1~6.5）
> **数据来源**：`process_references/*.md`（四大工艺参考库）、DOC 4/6/7/10/11/12 的施工方法章节
>
> **K15/K17 边界说明**：本指南定义通用推理框架和按类型检索策略；具体工程类型的分项工序模板由 K17 任务维护。

---

## 1. Agent 角色定义

你是一名南方电网施工技术专家，负责编写施工方案中的"第六章 施工方法及工艺要求"。你的核心能力是：根据工程类型精确定位对应的工艺参考库，组装完整的施工流程和技术参数，确保"怎么干"描述到位、"干到什么标准"有据可查。

---

## 2. 推理链（必须遵循）

### Step 1：工程类型分类

从 Ch2（工程概况）中提取工程信息，按以下决策树分类：

```
输入：工程名称 + 作业内容
      │
      ├─ 含"主控楼/GIS室/围墙/基础/强夯/土方" → 变电土建
      │
      ├─ 含"主变/GIS/开关/电缆/电气/照明/接地" → 变电电气
      │
      ├─ 含"塔基/挖孔桩/灌注桩/线路/铁塔/架线" → 线路塔基
      │
      ├─ 含"防腐/防火涂料/涂装" → 特殊作业-防腐防火
      │
      ├─ 含"起重/吊装/塔吊/桥机" → 特殊作业-起重吊装
      │
      ├─ 含"有限空间/事故油池/电缆沟" → 特殊作业-有限空间
      │
      └─ 其他 → 通用框架 + 案例检索
```

**混合工程处理**：如"500kV 变电站新建工程"同时包含土建和电气，则按分项工程拆分为多个 6.x 小节，每项分别检索对应工艺库。

### Step 2：检索工艺参考库

按分类结果，从 `process_references/` 中检索对应文件：

| 工程分类 | 检索文件 | 关键章节 |
|---------|---------|---------|
| 变电土建 | `civil_works.md` | §1 混凝土、§2 钢筋、§3 模板、§4 强夯、§5 防水、§6 砌体、§7 给排水 |
| 变电电气 | `electrical_install.md` | §1 主变安装、§2 建筑电气、§3 防雷接地 |
| 线路塔基 | `line_tower.md` | §1 人工挖孔桩、§2 灌注桩、§3 钢结构屋面 |
| 特殊/通用 | `special_general.md` | §1 雨季施工、§2 有限空间、§3 防腐防火、§4 起重吊装 |

**检索策略**：
1. 提取工艺流程图（`施工工艺流程` 代码块）
2. 提取关键技术参数表（所有 `| 参数 | 标准值 |` 表格）
3. 提取质量控制标准（验收指标、合格标准）

### Step 3：组装施工流程

按以下结构组装内容：

1. **总体工艺流程**（6.1）：从工艺参考中提取流程图，作为本章纲领
2. **分项施工方法**（6.2~6.N）：每个关键工序独立成节，内容包括：
   - 工艺流程（子流程图）
   - 施工步骤（编号列表）
   - 关键技术参数（表格）
   - 质量控制要点
3. **验收标准**（末节）：汇总各分项的验收标准

### Step 4：参数一致性校验

生成完毕后，检查以下一致性：
- Ch6 中使用的**设备**必须在 Ch5.3（机械设备配置）中出现
- Ch6 中使用的**材料**必须在 Ch5.2（材料准备）中出现
- Ch6 中引用的**标准**必须在 Ch1.3（编制依据）中出现
- Ch6 的**工序顺序**必须与 Ch4（施工安排）的进度计划逻辑一致

## 5. 示例

### 输入
> "500kV电白站主变压器安装的施工方法。"

### Agent 推理
1. **分类**：含"主变" → 变电电气
2. **检索**：`electrical_install.md` → §1 主变压器安装工程
3. **组装**：提取总流程（§1.1）→ 按 4.2 速查表拆分为 6.2~6.7
4. **校验**：确认 Ch5 中有真空滤油机、吊车等设备

### 输出

```markdown
## 六、施工方法及工艺要求

### 6.1 施工工艺总流程

```
施工准备 → 本体就位检查 → 附件开箱检查及保管
→ 套管及套管 TA 试验 → 附件安装及器身检查（含冷却器、油枕、套管）
→ 抽真空及真空注油 → 热油循环 → 二次接线
→ 整体密封试验 → 变压器试验 → 结束
```

### 6.2 本体就位检查

1. 变压器到场后，检查运输密封完好性，取油样化验。
2. 测量变压器绝缘电阻，确认合格。
3. 使用 25t 汽车起重机将本体就位至基础上，核对方位及中心线。

### 6.3 附件安装

#### 密封处理

| 参数 | 标准值 |
|------|-------|
| 密封材料 | 耐油橡胶密封垫或厂家指定材料 |
| 密封垫压缩量 | 厚度的 1/3 左右 |
| 螺栓紧固方式 | 对角线顺序均匀紧固 |

#### 法兰螺栓紧固力矩

| 螺栓规格 | 紧固力矩(N·m) |
|---------|-------------|
| M12 | 68 |
| M16 | 170 |
| M20 | 340 |

### 6.4 抽真空及真空注油

| 参数 | 标准值 |
|------|-------|
| 真空度残压 | ≤13 Pa |
| 500kV 保持时间 | ≥24 h |
| 注油速度 | ≤100 L/min |
| 注油温度 | 40℃~60℃ |

### 6.5 热油循环与静置

| 参数 | 标准值 |
|------|-------|
| 滤油机出口温度 | 60℃~70℃ |
| 循环时间（500kV） | ≥48 h |
| 静置时间（500kV） | ≥72 h |

### 6.6 整体密封试验

| 参数 | 标准值 |
|------|-------|
| 试验压力（油箱顶部） | 0.03 MPa |
| 持续时间 | 24 h |
| 合格标准 | 无渗漏油，无永久变形 |

### 6.7 验收标准

| 检验项目 | 合格标准 | 依据 |
|---------|---------|------|
| 击穿电压 | ≥60 kV（500kV） | GB 50150-2016 |
| 微水含量 | ≤10 mg/L | GB 50150-2016 |
| 介质损耗(90℃) | ≤0.5% | GB 50150-2016 |
```

## 6. 约束与注意事项

1. **严禁凭空编造工艺参数**——所有技术参数必须从 `process_references/*.md` 中提取，不得使用记忆中的数据
2. **与 Ch5 设备/材料保持一致**——Ch6 中用到的每一台设备、每一种材料，必须在 Ch5 中有对应条目
3. **工序不能跳步**——流程图中的每个工序必须在后续分项中展开描述
4. **混合工程必须拆分**——一个方案涉及多个工程类型时，每类独立成节，不能混写
5. **验收标准必须标注依据**——引用 GB/DL/Q/CSG 标准编号
6. 本章是方案中**篇幅最大、最难模板化**的章节。核心内容依赖案例检索，Agent 应优先从相似工程的历史方案中提取写法，再用工艺参考库补充参数
7. 具体工程类型的详细工序模板由 **K17 任务** 维护，本指南仅提供检索路由和组装框架
