## 1. 主变压器安装工程

### 1.1 总体施工工艺流程

```
施工准备 → 本体就位检查 → 附件开箱检查及保管
→ 套管及套管 TA 试验 → 附件安装及器身检查（含冷却器、油枕、套管）
→ 抽真空及真空注油 → 热油循环 → 二次接线
→ 整体密封试验 → 变压器试验 → 结束
```

### 1.2 本体检查与内检

**环境要求**：

| 条件 | 标准值 |
|------|-------|
| 天气 | 良好，防尘防雨防风可靠 |
| 本体温度 | 不低于周围环境温度 |
| 相对湿度 <75% 时器身暴露时间 | ≤16 h |
| 调压开关暴露时间 | ≤8 h |

**内检要点**：
1. 打开孔盖前必须通过排气孔平衡油箱内外压力
2. 检查铁芯及夹件紧固情况，铁芯接地绝缘电阻 ≥2MΩ
3. 检查绕组压板及垫块有无松动，引线绝缘是否完好
4. 彻底清理箱底残油及杂质

### 1.3 附件安装

**密封处理**：

| 参数 | 标准值 |
|------|-------|
| 密封材料 | 耐油橡胶密封垫或厂家指定材料 |
| 密封垫压缩量 | 厚度的 1/3 左右 |
| 螺栓紧固方式 | 对角线顺序均匀紧固 |

**法兰螺栓紧固力矩**：

| 螺栓规格 | 紧固力矩(N·m) |
|---------|-------------|
| M12 | 68 |
| M16 | 170 |
| M20 | 340 |

**套管安装**：
- 吊装前检查油位正常、瓷裙无损伤
- 使用专用吊具，保持倾斜角度与安装角度一致
- 套管末屏接地必须可靠，安装后做绝缘电阻测试

**冷却系统**：
- 安装前用合格变压器油冲洗，直至流出油与注入油无差别
- 油泵、风扇电机转向正确，运行无异音无振动

### 1.4 抽真空与真空注油（核心工艺）

**抽真空**：

| 参数 | 标准值 |
|------|-------|
| 真空度残压 | ≤13 Pa (0.1 Torr) |
| 500kV 保持时间 | ≥24 h |
| 抽真空时 | 监视箱壁变形情况 |

**真空处理时间**：

| 标称系统电压(kV) | 真空处理时间(h) |
|----------------|--------------|
| ≤69 | 12 |
| >69 且 <325 | 24 |
| ≥325 | 36 |

**注油**：

| 参数 | 标准值 |
|------|-------|
| 注油口 | 油箱底部阀门 |
| 注油速度 | ≤100 L/min（防静电） |
| 注油全程 | 保持真空状态 |
| 油温控制 | 40℃~60℃ |
| 注油至距箱顶 | 约 200 mm 时停止 |
| 停止后保持真空 | ≥4 h（补油至规定油面） |

### 1.5 热油循环

| 参数 | 标准值 |
|------|-------|
| 滤油机出口温度 | 60℃~70℃ |
| 油箱内最低温度 | ≥40℃ |
| 循环时间（500kV） | ≥48 h |
| 循环油量（500kV） | ≥3 倍变压器总油量 |

### 1.6 静置时间

| 电压等级 | 静置时间 |
|---------|---------|
| 500kV | ≥72 h |
| 220kV | ≥48 h |

### 1.7 整体密封试验

| 参数 | 标准值 |
|------|-------|
| 试验压力（油箱顶部） | 0.03 MPa |
| 持续时间 | 24 h |
| 合格标准 | 无渗漏油，无永久变形 |

### 1.8 油务指标（GB 50150-2016）

| 检验项目 | 500kV | 220kV |
|---------|-------|-------|
| 击穿电压 | ≥60 kV | ≥40 kV |
| 微水含量 | ≤10 mg/L | ≤15 mg/L |
| 介质损耗(90℃) | ≤0.5% | ≤0.5% |
| 含气量 | ≤1.0% | ≤1.0% |
| 界面张力(25℃) | ≥35 mN/m | ≥35 mN/m |
| 酸值 | ≤0.03 mgKOH/g | ≤0.03 mgKOH/g |

## 2. 建筑电气与照明工程

### 2.1 施工工艺流程

```
安装准备 → 管路敷设/桥架安装 → 管内穿线/电缆敷设
→ 灯具/开关/插座安装 → 配电箱安装 → 通电试运行
```

### 2.2 线管敷设

| 参数 | 标准值 |
|------|-------|
| 暗敷保护层厚度 | ≥15 mm |
| 消防管路保护层 | ≥30 mm |
| 弯曲半径（普通） | ≥6 倍管外径 |
| 弯曲半径（埋地/楼板） | ≥10 倍管外径 |
| 弯扁度 | ≤管外径的 10% |
| 镀锌钢管连接 | 螺纹连接 + 接地跨接 |
| PVC 管连接 | 套管粘接，牢固不漏浆 |

### 2.3 穿线与接线

**相色规定**：

| 相线 | 颜色 |
|------|------|
| A 相 | 黄色 |
| B 相 | 绿色 |
| C 相 | 红色 |
| N 线 | 淡蓝色 |
| PE 线 | 黄绿相间双色 |

**绝缘测试**：
- 穿线前清理管内积水杂物
- 穿线后、接终端前测试绝缘电阻
- 相间、相对地绝缘电阻 ≥0.5 MΩ

### 2.4 配电箱与终端

| 参数 | 标准值 |
|------|-------|
| 壁挂式底边距地高度 | 1.5 m |
| 垂直度偏差 | ≤1.5‰ |
| 同室开关/插座高度差 | ≤5 mm |
| 并列安装高度差 | ≤1 mm |
| 插座接线 | 单相两孔"左零右火"，三孔"左零右火上接地" |
| PE 线连接 | 严禁串联 |

## 3. 防雷与接地装置

### 3.1 接地体施工

**焊接搭接长度**：

| 材料 | 搭接长度 | 焊接要求 |
|------|---------|---------|
| 扁钢 | 宽度的 2 倍 | 至少三面施焊 |
| 圆钢 | 直径的 6 倍 | 双面施焊 |

**防腐处理**：焊接部位清除焊渣后涂刷沥青漆或防腐涂料

### 3.2 设备接地

| 项目 | 要求 |
|------|------|
| 主变接地点 | 2 点可靠接地 |
| 铁芯接地电流 | <100 mA |
| 等电位连接 | 金属管道、构架、电缆外皮均与接地干线连接 |
