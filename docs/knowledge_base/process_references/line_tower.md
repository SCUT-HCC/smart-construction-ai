## 1. 人工挖孔桩基础施工

### 1.1 施工工艺流程

```
测量分坑、场地清理 → 施工准备 → 孔口护壁制作
→ 孔口安全防护（软梯、安全绳、通风管线）→ 安装取土设备
→ 坑内气体检测 → 开挖土方 → 护壁制作 → 挖至设计桩深
→ 验孔 → 钢筋绑扎、支模 → 混凝土浇制及养护 → 拆模回填
```

### 1.2 土方开挖与护壁

| 参数 | 标准值 |
|------|-------|
| 每节开挖深度 | ≤1.0 m |
| 第一节护壁高出地面 | 100 mm（设锁口） |
| 护壁混凝土强度 | ≥C25 |
| 上下节搭接长度 | ≥50 mm |
| 竖向钢筋 | 上下两节必须有效搭接 |
| 土方运出 | 由专人提运至指定堆土场 |

### 1.3 通风与气体检测

| 项目 | 要求 |
|------|------|
| 检测时机 | 每日开工前 + 挖孔过程中每 2 小时 |
| 检测工具 | 毒气检测仪 |
| 深度阈值 | 孔深 >10 m 或有有害气体时必须强制通风 |
| 送风时间 | ≥5 min |
| 通风设备 | 鼓风机或风扇 |

### 1.4 钢筋笼安装

| 参数 | 标准值 |
|------|-------|
| 分段制作 | 钢筋笼过长时分段，入孔后焊接 |
| 临时加劲架间距 | 3~4 m（安放后拆除） |
| 吊装方式 | 按规定吊点位置绑扎 |
| 禁止行为 | 孔底未清理干净时严禁强行下压 |

### 1.5 混凝土浇筑

**干式浇筑**（孔底积水 ≤50 mm）：

| 参数 | 标准值 |
|------|-------|
| 落距 >3 m 时 | 必须使用串筒 |
| 串筒末端距孔底 | ≤2 m |
| 浇筑方式 | 通过溜槽 |

**水下浇筑**（积水较多且无法抽干）：
- 采用导管法（水下混凝土灌注法）
- 首批灌注量须满足导管初次埋置深度要求
- 导管直径一般 250~300 mm

## 2. 灌注桩基础施工

### 2.1 施工工艺流程

```
测量定位 → 场地平整 → 护筒埋设 → 钻机就位
→ 钻孔（循环泥浆护壁）→ 清孔 → 终孔验收
→ 钢筋笼制作与安装 → 混凝土灌注 → 拔出护筒 → 养护
```

### 2.2 关键参数（参考 DOC 12）

**桩基参数范围**：

| 参数 | 典型值 |
|------|-------|
| 桩径 | 1200~2400 mm |
| 桩长 | 13500~24500 mm |
| 埋深 | 12000~22000 mm |
| 外露 | 1500 mm |
| 桩身混凝土强度 | C35 |
| 承台垫层强度 | C20 |

**混凝土配合比（C35）**：

| 水胶比 | 水泥(kg/m³) | 水(kg/m³) | 砂(kg/m³) | 石(kg/m³) | 坍落度(mm) |
|-------|-----------|---------|---------|---------|-----------|
| 0.55 | 400 | 220 | 748 | 1032 | 180~220 |

### 2.3 基础预偏设计（转角塔）

| 转角度数 α | 预偏值 Δh |
|-----------|----------|
| 0° < α ≤ 3° | 3B/1000 |
| 3° < α ≤ 20° | 4B/1000 |
| 20° < α ≤ 40° | 8B/1000 |
| 40° < α ≤ 60° | 10B/1000 |
| 60° < α ≤ 90° 及终端 | 12B/1000 |

> B = 基础根开值

### 2.4 导线对地最小距离（500kV）

| 地区类型 | 最小距离(m) | 导线状态 |
|---------|-----------|---------|
| 居民区 | 14 | 40℃弧垂 |
| 非居民区 | 11 | 40℃弧垂 |
| 交通困难地区 | 8.5 | 40℃弧垂 |
| 建筑物（垂直） | 9 | 40℃弧垂 |
| 树木（垂直） | 7.0 | 40℃弧垂 |

### 2.5 导线对交叉跨越物最小距离（500kV）

| 跨越物 | 最小距离(m) | 导线状态 |
|-------|-----------|---------|
| 高速公路 | 14.0 | 70℃弧垂 |
| 铁路轨顶 | 14.0 | 70℃弧垂 |
| 通航河流 | 9.5 | 40℃弧垂 |
| 电力线 | 6.0 | 40℃弧垂 |
| 特殊管道 | 7.5 | 40℃弧垂 |

## 3. 钢结构屋面工程（GIS 室楼承板）

### 3.1 施工工艺流程

```
准备工作 → 楼承板（桁架）安装固定 → 纵向钢筋绑扎
→ 混凝土泵送浇筑 → 混凝土找平 → 混凝土养护
```

### 3.2 压型钢板安装

| 参数 | 标准值 |
|------|-------|
| 在钢梁上支承长度 | ≥50 mm |
| 侧向连接 | 咬口钳连接 |
| 端部锚固 | 熔焊连接 |
| 焊点直径 | ≥20 mm |
| 焊点间距 | ≤300 mm |

**开孔处理**：
- 尺寸 <500×500 mm：浇筑后用切割机切除，无需加强
- 尺寸 >500 mm：按设计要求加设洞口边梁

### 3.3 栓钉（剪力钉）焊接

| 参数 | 要求 |
|------|------|
| 焊前处理 | 清除钢梁表面油漆、铁锈 |
| 焊接设备 | 专用栓钉焊机 |
| 外观检查 | 焊脚均匀，无气孔夹渣裂纹 |
| 弯曲试验 | 弯曲 30° 后焊缝无裂纹 |

### 3.4 屋面混凝土浇筑

| 参数 | 标准值 |
|------|-------|
| 浇筑方向 | 垂直于压型钢板波纹方向 |
| 振捣工具 | 平板振动器或小直径插入式振捣器 |
| 振捣注意 | 不得直接接触压型钢板 |
| 找平工具 | 刮尺刮平 + 木抹子搓平 |
| 二次抹压 | 初凝前进行，防止表面裂缝 |
| 养护时间 | ≥7 天 |
