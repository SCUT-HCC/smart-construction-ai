## 1. 雨季施工及防台防汛措施

### 1.1 施工原则

| 原则 | 要求 |
|------|------|
| 防雨 | 小雨不停工，大雨大风停止室外施工 |
| 排水 | 施工场地做好排水沟规划，规划排水分区 |
| 防护 | 露天机械搭设防晒防雨防砸安全棚 |
| 持证 | 特殊工种操作人员必须持证上岗 |

### 1.2 混凝土工程雨季措施

| 环节 | 措施 |
|------|------|
| 浇筑前 | 检查脚手架牢固性和防滑措施；掌握天气情况 |
| 坍落度 | 及时测定砂石含水率，调整配合比 |
| 覆盖 | 混凝土未凝时严禁雨水冲刷，准备足够覆盖材料 |
| 施工缝 | 合理预留，调整工作时间 |

### 1.3 钢筋及焊接雨季措施

| 环节 | 措施 |
|------|------|
| 堆放 | 场地平整排水通畅，放垫木，雨前覆盖塑料布 |
| 焊接环境 | 雨天无防雨措施时禁止焊接，接头焊后不能立即接触雨水 |
| 焊条管理 | 受潮焊条须烘干：碱性 300~350℃，酸性 100~150℃ |
| 焊条存储 | 烘焙后存于 100℃ 恒温箱，随用随取 |

### 1.4 临时用电与机具防护

| 项目 | 要求 |
|------|------|
| 配电箱 | 上锁，有防雨盖板，有危险标志 |
| 电线 | 禁止使用破损或绝缘不良电线，禁止电线随地走 |
| 漏电保护 | 所有机具装有漏电保护装置 |
| 配电制度 | "一机、一闸、一漏、一箱" |
| 照明 | 基坑内外必须配备足够照明灯，设围栏和安全标识 |

### 1.5 防台防风措施

| 场景 | 措施 |
|------|------|
| 塔吊 | 吊钩升至最高限位以下，小车收至臂根，大臂顺风自由回转 |
| 脚手架 | 检查并加固连墙件 |
| 临时建筑 | 加固板房，门窗关闭 |
| 排水系统 | 提前疏通，准备应急排水泵 |
| 断电 | 切断非必要用电设备电源 |

## 2. 有限空间作业安全措施

### 2.1 作业原则（七不准）

1. 未经风险辨识**不准**作业
2. 未经通风和检测合格**不准**作业
3. 不佩戴劳动防护用品**不准**作业
4. 没有监护**不准**作业
5. 电气设备不符合规定**不准**作业
6. 未经审批**不准**作业
7. 未经培训演练**不准**作业

**总原则**：先通风 → 再检测 → 后作业

### 2.2 通风要求

| 参数 | 标准值 |
|------|-------|
| 横断面平均风速 | ≥0.8 m/s |
| 或换气次数 | ≥20 次/h |
| 单出入口 | 送风口置于作业区域 |
| 多出入口 | 近处送风，远处排风 |
| 死角处理 | 设挡板或改变吹风方向 |

### 2.3 气体检测

**最低检测三项**：

| 检测指标 | 安全范围 |
|---------|---------|
| 氧浓度 | 19.5%~23.5% |
| 可燃气体浓度 | <最低爆炸极限的 10% |
| 一氧化碳浓度 | <25 ppm |

**检测要求**：

| 参数 | 标准值 |
|------|-------|
| 检测频率 | 环境变化时持续或定时检测 |
| 检测点数量（高度 ≤8m） | ≥2 个 |
| 上检测点 | 距顶部 ≤1 m |
| 下检测点 | 距底部 ≤1 m |
| 检测记录 | 包含时间、地点、气体种类、浓度 |

### 2.4 现场监护与应急

| 项目 | 要求 |
|------|------|
| 监护人员 | 专职监护，保持与作业人员联系，不得离岗 |
| 应急装备 | 三脚架、安全绳（全身式安全带）、正压式空气呼吸器、气体检测仪、防爆对讲机、急救箱 |
| 紧急撤离 | 检测仪报警或人员不适时立即停止作业并撤离 |

## 3. 防腐与防火涂料施工

### 3.1 表面处理（关键工序）

| 参数 | 标准值 |
|------|-------|
| 除锈等级 | Sa2.5（喷砂/抛丸） |
| 除锈后底漆涂装 | 4 小时内 |
| 禁止作业条件 | 相对湿度 >85% 或钢材表面温度低于露点 3℃ |
| 表面要求 | 无可见油脂、污垢、氧化皮、铁锈和旧涂层 |

### 3.2 构支架防腐涂装

**涂层体系**：

| 涂层 | 涂料类型 | 干膜厚度 |
|------|---------|---------|
| 底漆 | 环氧富锌底漆 | ≥50μm |
| 中间漆 | 环氧云铁中间漆 | ≥100μm |
| 面漆 | 聚氨酯面漆或氟碳面漆 | ≥50μm |
| **总厚度** | — | **≥200μm** |

**涂装操作**：

| 参数 | 标准值 |
|------|-------|
| 施工方式 | 高压无气喷涂 |
| 喷枪与工件距离 | 250~350 mm |
| 喷枪角度 | 垂直于工件表面 |
| 移动速度 | 均匀 |
| 涂层间隔 | 上一道干燥后方可涂下一道 |
| 质量要求 | 平整光滑，色泽一致，无流挂起泡剥落 |

### 3.3 钢结构防火涂料

| 参数 | 标准值 |
|------|-------|
| 基层处理 | 防锈底漆涂装完毕并验收合格 |
| 每层喷涂厚度（薄型） | ≤3 mm |
| 每层喷涂厚度（厚型） | 5~8 mm |
| 第一遍 | 以盖住基层为准，防止流坠 |
| 层间间隔 | 前遍表干后（4~24 小时） |
| 裂纹宽度（厚型） | ≤1 mm |
| 质量要求 | 无空鼓、无脱落 |
| 厚度要求 | 满足设计耐火极限 |

## 4. 起重吊装工程

### 4.1 桥式起重机安装流程

```
基础验收 → 组装轨道/行走机构 → 安装主梁/端梁
→ 安装电气系统 → 安装起升机构 → 试运转 → 验收
```

### 4.2 塔式起重机安装流程

```
选定位置 → 浇筑承台基础（预埋螺栓）→ 安装基础节
→ 安装顶升套架 → 安装过渡节 → 安装回旋总成
→ 安装平衡臂 → 安装起重臂 → 安装配重
→ 电气接线、安装钢丝绳 → 塔机试运转
→ 顶升塔身 → 整机调试
```

### 4.3 塔吊拆除流程

```
拆卸平衡重（保留一块）→ 拆卸起重臂 → 拆卸剩余平衡重
→ 拆卸平衡臂 → 拆卸驾驶室及回转总成
→ 拆卸标准节及附墙装置 → 拆卸底架及基础节
```

### 4.4 顶升加节安全要点

| 项目 | 要求 |
|------|------|
| 平衡控制 | 上部重心落在顶升油缸轴线上 |
| 禁止操作 | 顶升过程中禁止回转、起升、变幅 |
| 风力限制 | 四级以上大风禁止顶升 |
| 螺栓紧固 | 每加完一节标准节立即紧固连接螺栓 |

### 4.5 吊装安全通用要求

| 项目 | 要求 |
|------|------|
| 吊装计算 | 重量、重心、吊点、绳索强度须经计算 |
| 试吊 | 正式吊装前进行试吊（离地 200~300mm，悬停 10min） |
| 指挥 | 专人指挥，统一信号 |
| 禁区 | 吊臂回转范围内设警戒区，非作业人员不得入内 |
| 风力限制 | 六级以上大风禁止高处吊装作业 |

## 5. 绿色施工与环境保护措施

### 5.1 扬尘控制（"六个 100%"）

| 措施 | 要求 |
|------|------|
| 围挡 | 连续封闭，市区 ≥2.5m，一般 ≥1.8m |
| 裸土覆盖 | 超过 48h 不施工的裸土必须覆盖 |
| 车辆冲洗 | 出入口设冲洗设施，建冲洗台账，净车出场 |
| 洒水降尘 | 土方作业必须洒水喷雾；四级以上大风停止土方作业 |
| 道路硬化 | 施工主要道路硬底化处理 |
| 密闭运输 | 运输车辆密闭或覆盖 |

### 5.2 噪音与光污染

| 项目 | 要求 |
|------|------|
| 夜间施工禁令 | 22:00~次日 6:00 禁止产生噪声污染的施工 |
| 特殊连续作业 | 办理夜间施工许可证并公告居民 |
| 设备降噪 | 选用低噪声设备，搭建防噪棚 |
| 光污染 | 夜间照明加灯罩，集中透光方向 |
| 电焊遮挡 | 采取遮挡措施，防止弧光外泄 |

### 5.3 水资源与排放

| 项目 | 要求 |
|------|------|
| 循环用水 | 设循环水系统，雨水/基坑水用于降尘、冲洗、灌溉 |
| 生活污水 | 厕所设化粪池，食堂设隔油池 |
| 生产废水 | 经沉淀池达标后方可排入市政管网 |

## 6. 文明施工与标准化

### 6.1 7S 管理实施要点

| 要素 | 实施要点 |
|------|---------|
| 整理 | 区分要与不要，清除无用物品 |
| 整顿 | 定置管理，物料摆放整齐有序，挂标识牌 |
| 清扫 | 每日"工完料尽场地清" |
| 安全 | 消除隐患，确保本质安全 |
| 节约 | 杜绝浪费，水电随手关 |

### 6.2 标识标牌标准化

| 项目 | 内容 |
|------|------|
| 五牌一图 | 工程概况牌 + 管理人员名单牌 + 消防保卫牌 + 安全生产牌 + 文明施工牌 + 施工平面布置图 |
| 安全警示 | 孔洞临边、高压线、吊装区悬挂国标安全警示标志 |
| 分区标识 | 办公区、生活区、作业区分开设置 |
| 高压区域 | 设定型化围栏，挂"止步，高压危险"标示牌 |
