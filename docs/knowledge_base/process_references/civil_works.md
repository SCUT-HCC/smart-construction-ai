## 1. 混凝土工程

### 1.1 浇筑工艺

| 参数 | 标准值 | 说明 |
|------|-------|------|
| 自由倾落高度 | ≤2 m | 超过时必须使用串筒或溜槽 |
| 分层浇筑厚度（插式振动器） | ≤500 mm | — |
| 分层浇筑厚度（平板振动器） | ≤200 mm | — |
| 振动棒移动间距 | ≤1.5R | R = 振动器作用半径，约 300~400 mm |
| 振动棒插入下层深度 | ≥50 mm | 确保层间结合 |
| 每点振捣时间 | 15~30 s | 以表面泛浆、不再冒泡为准 |
| 混凝土抗压强度起步值 | 1.2 MPa | 达到前不得踩踏或施工 |

### 1.2 养护要求

| 参数 | 标准值 |
|------|-------|
| 覆盖浇水起始时间 | 浇筑后 12h 内 |
| 普通混凝土养护时间 | ≥7 天 |
| 掺缓凝剂/抗渗混凝土 | ≥14 天 |
| 养护方式 | 覆盖保湿膜 + 浇水 |

### 1.3 冬/雨季措施

**雨季浇筑**（参考 DOC 8）：
- 浇筑前检查脚手架牢固性和防滑措施
- 砂石含水率变化时须及时调整配合比，严格控制坍落度
- 混凝土未凝固时严禁雨水冲刷，现场必须有足够覆盖材料

**冬季浇筑**：
- 混凝土入模温度不低于 5℃
- 及时覆盖保温，防止早期冻害

## 2. 钢筋工程

### 2.1 加工与绑扎

| 参数 | 标准值 |
|------|-------|
| 钢筋接头检验合格率 | ≥95% |
| 每楼层或 300 个接头抽检 | 1 组焊接接头 |
| 梁顶面主筋净间距 | ≥30 mm（利于混凝土灌筑） |
| 保护层垫块间距 | ≤1000 mm |
| 双层排列间距垫筋直径 | 25 mm |

### 2.2 焊接要求

**电渣压力焊**：
- 适用于竖向钢筋连接
- 焊接面必须清除铁锈、油污

**手工电弧焊**：
- 搭接焊缝长度：单面焊 ≥10d，双面焊 ≥5d（d 为钢筋直径）
- 焊缝宽度应大于钢筋直径的 0.7 倍

### 2.3 雨季钢筋措施（参考 DOC 8）

- 钢筋堆放场地排水通畅，下放垫木，雨前覆盖塑料布
- 焊接设备注意防潮防雨，雨天无防雨措施时禁止焊接
- 焊条受潮须烘干：碱性低氢型 300~350℃，酸性型 100~150℃
- 烘焙后存储在 100℃ 恒温箱内随用随取

## 4. 强夯地基处理

### 4.1 施工工艺流程

```
场地平整 → 测量放线（标定夯点） → 第一遍点夯 → 推平夯坑
→ 第二遍点夯 → 推平夯坑 → 第一遍满夯 → 第二遍满夯 → 整平碾压 → 检测验收
```

### 4.2 关键参数（参考 DOC 11）

| 参数 | 设计值 |
|------|-------|
| 处理面积 | 18404 m² |
| 点夯单击能 | 6000 kN·m |
| 满夯单击能 | 4000 kN·m |
| 夯锤重量 | 32 t |
| 夯锤直径 | 2.2 m |
| 夯锤落距 | 15.5 m |
| 夯点布置 | 正三角形，间距 6.0 m |
| 夯击遍数 | 点夯 2 遍（隔行），满夯 2 遍 |
| 强夯设备 | BWQH 6500 强夯机 |

### 4.3 质量控制标准

| 指标 | 目标值 |
|------|-------|
| 压实系数 | ≥0.94 |
| 地基承载力特征值 | ≥180 kPa |
| 压缩模量 Es | ≥6.5 MPa |

### 4.4 收锤标准

- 最后两击平均夯沉量不大于设计规定值（一般 50~100 mm）
- 夯坑周围无明显隆起
- 夯击次数达到设计要求
