# 岗位职责模板库

> Agent 在编写"三、施工组织机构及职责"时从本文件检索通用岗位职责。
> 所有数据均来自实际施工方案（DOC 6 质量/安全管理章节、DOC 7 施工组织机构章节）。
>
> **最后更新**：2026-02-23

---

## 1. 标准组织架构（大型工程 ≥500kV）

```
              项目经理
                │
        ┌───────┼───────┐
        │       │       │
      项目总工  项目副经理  ──→ 综合部
        │
  ┌─────┼─────┐
  │     │     │
工程部  质管部  安监部
  │
施工队/班组
```

## 2. 关键岗位职责

### 2.1 项目经理

| 序号 | 职责内容 |
|------|---------|
| 1 | 工程质量、安全施工的第一责任人，对工程质量和安全负全面领导责任 |
| 2 | 贯彻执行国家、行业法律法规和企业各项管理规定 |
| 3 | 制定工程质量目标和安全目标，确保有效贯彻实施 |
| 4 | 组织编制施工组织设计、质量计划、安全技术措施等管理文件 |
| 5 | 建立健全质量管理体系和安全管理体系，确保有效运行 |
| 6 | 对生产要素（人、材料、机械、方法、环境）进行优化配置和动态管理 |
| 7 | 定期组织质量检查和安全检查，参加重要验收及评优工作 |
| 8 | 按规定上报质量事故和安全事故，组织事故调查处理，制定预防措施 |
| 9 | 保证安全技术措施经费提取和使用，确保专款专用 |
| 10 | 在计划、布置、检查、总结、评比施工任务的同时，把安全工作贯穿到每个施工环节 |

> 来源：DOC 6 第 7.2.2 节、8.2.2 节

### 2.2 项目副经理

| 序号 | 职责内容 |
|------|---------|
| 1 | 在项目经理领导下，做好分管范围内的质量管理和安全管理 |
| 2 | 协助项目经理开展工程管理工作，保证质量和安全活动受控 |
| 3 | 项目经理不在现场时，履行项目经理管理职责 |
| 4 | 参加安全大检查、安全事故调查分析处理工作 |

### 2.3 项目总工（技术负责人）

| 序号 | 职责内容 |
|------|---------|
| 1 | 在项目经理领导下，贯彻执行质量方针，在技术领域落实质量和安全体系文件 |
| 2 | 参与制定工程质量目标，协助项目经理的各项领导工作 |
| 3 | 参加图纸会审，主持施工交底，主持编制施工组织设计 |
| 4 | 审查施工技术方案、施工组织措施和施工作业指导书 |
| 5 | 主持技术质量工作，处理重大技术质量问题 |
| 6 | 定期召开质量管理工作会议，掌握质量信息和动态，分析趋势 |
| 7 | 指导工程部开展技术管理工作，督促技术人员进行工艺交底 |
| 8 | 负责施工组织设计中安全施工、文明施工措施的编制 |

> 来源：DOC 6 第 7.2.2 节、8.2.2 节

## 3. 部门职责

### 3.1 工程部

| 序号 | 职责内容 |
|------|---------|
| 1 | 负责施工全过程控制，为施工全过程受控提供技术保证 |
| 2 | 负责技术性文件、质量体系文件和资料以及工程移交资料的控制管理 |
| 3 | 编写和审核各种施工方案和施工技术措施、技术文件 |
| 4 | 检查和监督各种技术文件、施工方案和技术措施的实施 |
| 5 | 负责图纸审核和技术交底工作，参与原材料质量检查验收 |
| 6 | 负责组织重要施工工序的安全施工措施交底 |
| 7 | 组织编制新技术、新工艺中的安全施工措施 |

### 3.2 质管部

| 序号 | 职责内容 |
|------|---------|
| 1 | 负责质量保证体系的综合管理工作，确保正常运转 |
| 2 | 具体组织执行质量计划及质量保证措施 |
| 3 | 深入施工现场，监督检查工程质量，对问题及时提出建议 |
| 4 | 积极配合建设单位、监理工程师的质量检查，跟踪整改措施落实 |
| 5 | 组织开展 QC 小组活动等质量改进活动 |
| 6 | 定期组织质量会议，对工程质量进行分析 |

### 3.3 安监部

| 序号 | 职责内容 |
|------|---------|
| 1 | 负责日常安全管理工作，确保安全管理体系正常运转 |
| 2 | 制定安全管理制度和安全技术措施 |
| 3 | 审查施工组织设计、施工技术方案中的安全施工措施，监督执行 |
| 4 | 深入施工现场，掌握安全施工动态，监督控制安全条件和作业行为 |
| 5 | 组织定期安全会议，监督检查安全员工作 |
| 6 | 协助组织安全大检查，按"三定"原则（定人、定时间、定项目）督促整改 |
| 7 | 参加安全事故调查处理，负责事故统计、分析和上报 |

### 3.4 综合部

| 序号 | 职责内容 |
|------|---------|
| 1 | 负责设备、材料进货检验，参与甲供设备开箱检验 |
| 2 | 做好设备、材料的储存、保管、标识、防护、运输等工作 |
| 3 | 负责施工机具的采购、调配、保养、管理 |
| 4 | 负责对分包方进行资格评审，对外包合同进行管理 |
| 5 | 保证安全技术措施经费提取和使用，做到专款专用 |
| 6 | 负责后勤保障工作，包括现场保卫、消防、卫生保健 |

## 4. 一线人员职责

### 4.1 质检员

| 序号 | 职责内容 |
|------|---------|
| 1 | 执行落实各级下达的有关质量文件和指令 |
| 2 | 在施工中执行国家标准和技术规范，执行施工方案和技术措施 |
| 3 | 负责施工工艺、质量管理及技术指标在现场的全面贯彻实施 |
| 4 | 深入施工现场，严格按标准进行检查验收 |
| 5 | 保证质量记录完整、齐全、真实 |
| 6 | 收集原始质量记录，整理移交资料 |

### 4.2 安全员

| 序号 | 职责内容 |
|------|---------|
| 1 | 深入施工现场，监督检查安全状况和施工人员作业行为，制止违章作业 |
| 2 | 参加重要和危险性作业项目开工前的安全交底，检查开工安全条件 |
| 3 | 参加安全检查，按"三定"原则督促整改 |
| 4 | 监督检查班组每天"三交"（交任务、交安全、交技术）和"三查"（查衣着、查"三宝"、查精神状态） |
| 5 | 对施工现场安全设施进行检查，确保完好有效 |
| 6 | 组织开展安全教育工作，负责二级安全教育 |
| 7 | 按"四不放过"原则协助事故调查处理 |

### 4.3 施工队组长

| 序号 | 职责内容 |
|------|---------|
| 1 | 本队安全施工第一责任者，合理组织安排施工班组 |
| 2 | 组织工程施工质量自检、互检，确保不合格产品不转入下一道工序 |
| 3 | 定期召开队内施工质量会议和安全日活动 |
| 4 | 执行施工工艺纪律，杜绝质量通病 |
| 5 | 开工前组织人员接受安全交底，对危险作业安排安全监护人 |

## 5. 工程规模与人员配置推荐（需确认）

| 工程规模 | 典型工程 | 管理人员 | 安全员 | 质检员 | 施工队 |
|---------|---------|---------|--------|--------|--------|
| 大型（500kV 新建） | 变电站新建工程 | 项目经理 + 副经理 + 总工 + 副总工 | ≥2 人 | ≥2 人 | 3~5 个 |
| 中型（220kV 或专项） | 单体建筑/主变安装 | 项目经理 + 总工 | ≥1 人 | ≥1 人 | 1~3 个 |
| 小型（110kV 或专题） | 防腐/起重/有限空间 | 工作负责人 + 技术负责人 | 1 人 | 兼任 | 1 个 |

> **注意**：以上配置为基于 16 份样本文档的推荐值，具体配置需根据工程实际情况和南网相关管理规定确定。
