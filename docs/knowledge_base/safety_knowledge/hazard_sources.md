# 危险源辨识与风险评估

> Agent 在编写"八、安全文明施工管理"时从本文件检索工程类型对应的危险源清单。
> 所有数据均来自实际施工方案（DOC 6 第 8.4.1 节、DOC 7/9/10 安全管理章节）。
>
> **最后更新**：2026-02-23

---

## 1. 风险等级定义

| 等级 | 分值 | 含义 | 控制要求 |
|------|------|------|---------|
| 5 | 极高风险 | 可能导致群体伤亡或重大财产损失 | 编制专项方案，专家论证，全程旁站监护 |
| 4 | 高风险 | 可能导致人身重伤 | 编制专项安全措施，设专人监护 |
| 3 | 中等风险 | 可能导致人身轻伤或设备损坏 | 纳入日常安全管理，定期检查 |
| 2 | 低风险 | 可能导致轻微伤害 | 常规安全措施即可 |
| 1 | 极低风险 | 几乎不会造成伤害 | 提示注意 |

## 6. 事故类型统计（用于 Ch9 应急预案选型）

> 按以下类型覆盖本工程可能的事故场景：

| 事故类型 | 典型场景 | 适用工程 |
|---------|---------|---------|
| 高处坠落 | 脚手架作业、临边"四口"、塔上作业 | 全部 |
| 物体打击 | 吊装、孔口坠物、抛掷物体 | 全部 |
| 坍塌 | 模板支撑失稳、基坑坍塌、孔壁坍塌 | 土建、线路 |
| 触电 | 临时用电、高压试验、带电作业附近 | 全部 |
| 机械伤害 | 钻机/圆锯/提升机操作不当 | 全部 |
| 火灾 | 油务操作、焊接、易燃品 | 电气、土建 |
| 中毒窒息 | 有限空间、涂装作业 | 有限空间、防腐 |
| 起重伤害 | 超载、吊具失效、信号不明 | 起重吊装 |
