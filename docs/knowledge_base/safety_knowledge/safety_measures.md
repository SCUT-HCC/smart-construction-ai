## 1. 安全保证措施（按场景分类）

### 1.1 施工现场通用措施

| 序号 | 措施内容 |
|------|---------|
| 1 | 按施工总平面图布置生产、生活临时设施、材料堆场 |
| 2 | 砌筑围墙，设置门卫，封闭管理 |
| 3 | 进入施工现场必须正确佩戴安全帽 |
| 4 | 保持整洁，"工完、料尽、场地清" |
| 5 | 机动车限速行驶，供电线路下设限高标志 |
| 6 | 夜间施工有足够照明 |
| 7 | 悬挂宣传标志、警告标志 |

### 1.2 消防措施

| 序号 | 措施内容 |
|------|---------|
| 1 | 组建义务消防组织，定期消防培训和演习 |
| 2 | 按规定配置足够消防器材 |
| 3 | 加强易燃易爆物品管理，严禁在办公/宿舍存放 |
| 4 | 模板加工场、仓库严禁吸烟，设"严禁烟火"标志 |
| 5 | 电焊/火焊时做好防火灭火措施 |

### 1.3 临时用电安全

| 序号 | 措施内容 |
|------|---------|
| 1 | 合格电工敷设线路和安装设备，严禁非电气人员操作 |
| 2 | 采用 TN-S（三相五线制）接零保护系统 |
| 3 | 用电设备金属外壳装设接地保护 |
| 4 | 配电箱安装漏电保护器和熔断器，严禁替代熔丝 |
| 5 | 严禁电线直接勾挂闸刀或插入插座 |
| 6 | "一机、一闸、一漏、一箱"制度 |
| 7 | 配电箱上锁、有防雨盖板、有危险标志 |

### 1.4 高处作业与交叉作业

| 序号 | 措施内容 |
|------|---------|
| 1 | 平台走道装设防护栏杆和挡脚板，或设防护立网 |
| 2 | 孔洞、沟道设盖板或围栏 |
| 3 | 夜间高处作业有足够照明 |
| 4 | 正确使用安全带，穿软底鞋，使用工具袋 |
| 5 | 交叉作业设防护隔离设施 |
| 6 | 交叉作业信道畅通，危险出入口设围栏或警告牌 |

### 1.5 季节性施工

| 序号 | 措施内容 |
|------|---------|
| 1 | 夏季做好防暑降温 |
| 2 | 气温过高时调整作息时间，避开高温时段 |
| 3 | 注意饮食卫生，防流行传染病 |
| 4 | 雨季：小雨不停工，大雨大风停止室外施工 |
| 5 | 雨季：排水沟规划，机械搭设防雨棚 |
| 6 | 台风：塔吊顺风自由回转，脚手架加固连墙件 |

> 来源：DOC 6 第 8.4.2 节"施工安全保证措施一览表"

## 2. 安全管理制度（通用模板）

### 2.1 三级安全教育

| 级别 | 教育内容 | 负责人 |
|------|---------|-------|
| 一级（公司级） | 国家安全法规、企业安全制度、典型事故案例 | 公司安全部门 |
| 二级（项目级） | 现场安全规章、工程概况、危险源辨识 | 项目安监部/安全员 |
| 三级（班组级） | 岗位操作规程、安全注意事项、应急措施 | 班组长 |

### 2.2 安全检查制度

| 检查类型 | 频率 | 参加人员 |
|---------|------|---------|
| 定期安全大检查 | 每月 1 次 | 项目经理、总工、安监部 |
| 专项安全检查 | 按需 | 安监部、相关专业人员 |
| 日常巡查 | 每日 | 安全员 |
| 班前安全交底 | 每班次 | 班组长 |

### 2.3 安全活动制度

| 活动 | 频率 | 内容 |
|------|------|------|
| 安全日活动 | 每周 1 次 | 安全学习、事故分析、经验交流 |
| 站班会 | 每日 | "三交"（交任务/安全/技术）+ "三查"（查衣着/三宝/精神状态） |
| 安全会议 | 每周/每月 | 安全工作布置、检查、总结 |

## 3. 文明施工措施

### 3.1 管理措施

| 序号 | 措施内容 |
|------|---------|
| 1 | 成立安全生产领导小组，设专职安全员 |
| 2 | 进场员工三级安全教育培训，考试合格上岗，"三无"人员禁止进场 |
| 3 | 贯彻"安全第一，预防为主"方针 |
| 4 | 明确文明施工目标，制度化、规范化、标准化 |
| 5 | 每周"安全日"活动，有内容、有目的、有记录 |
| 6 | 施工前先交底，严格执行操作规程 |

### 3.2 现场标准化

| 项目 | 要求 |
|------|------|
| 施工标牌 | 醒目位置设施工标牌（五牌一图） |
| 安全防护 | 开挖面两侧搭设 1.2m 高防护栏杆，涂刷警示漆 |
| 危险源标识 | 设立危险源辨识牌 |
| 材料管理 | 严禁上下抛掷，使用工具袋或料框 |
| 机具管理 | 清洁，经常维护，设漏电保护 |
| 垃圾处理 | 及时清理，拆除模板/脚手架统一堆放 |

### 3.3 7S 管理要点

| 要素 | 实施要点 |
|------|---------|
| 整理 | 区分要与不要，清除无用物品 |
| 整顿 | 定置管理，物料整齐有序，挂标识牌 |
| 清扫 | 每日"工完料尽场地清" |
| 清洁 | 保持施工环境整洁 |
| 素养 | 遵守规章制度，养成良好习惯 |
| 安全 | 消除隐患，确保本质安全 |
| 节约 | 杜绝浪费，水电随手关 |

## 4. 环境保护措施（Ch8.6 内容）

> 注意：此处为 Ch8.6 的即时性环保措施（"不出事"），与 Ch10 的系统性绿色施工管理（"做得更好"）有本质区别。

| 序号 | 措施内容 |
|------|---------|
| 1 | 加强环保宣传教育，增强环保意识 |
| 2 | 材料堆放整齐，定量供应，工完料尽场地清 |
| 3 | 天气干燥时路面洒水防扬尘；做好排水沟防积水 |
| 4 | 生活垃圾定点放置，设专人管理，最多两天清理一次 |
| 5 | 不可利用材料按指定地点弃置，弃渣场设排水系统 |
| 6 | 保护周围林草和水土保持设施，减少地表扰动 |
