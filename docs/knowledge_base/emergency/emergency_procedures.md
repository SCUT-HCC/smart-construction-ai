# 应急处置程序库

> Agent 在编写"九、应急预案与处置措施"时从本文件检索各类事故的应急处置程序。
> 所有数据均来自实际施工方案（DOC 6 第 8.4.3 节、DOC 9 有限空间应急章节）及南网应急管理标准。
>
> **最后更新**：2026-02-23

## 1. 应急响应通用流程

```
发现事故/险情 → 立即报警（现场人员）
→ 启动应急预案（应急指挥机构）
→ 现场急救（应急救援队）
→ 疏散人员（现场负责人）
→ 控制事态（应急指挥）
→ 上报上级（项目经理 → 公司 → 业主/监理）
→ 事故调查与处理
→ 总结改进
```

### 1.1 报告流程

| 步骤 | 时限 | 负责人 | 内容 |
|------|------|--------|------|
| 现场报告 | 立即 | 发现人/现场负责人 | 事故类型、位置、伤亡情况 |
| 项目部报告 | 30min 内 | 项目经理 | 启动应急响应，通知相关部门 |
| 公司报告 | 1h 内 | 项目经理 | 事故详情，初步判断，已采取措施 |
| 业主/监理报告 | 2h 内 | 项目经理 | 事故经过，处置进展 |
| 政府部门报告 | 按规定 | 公司安全部门 | 重大及以上事故需报政府部门 |

### 1.2 应急组织架构（通用模板）

```
应急总指挥（项目经理）
├── 副总指挥（项目总工/副经理）
├── 抢险救援组：现场救援、设备抢修
├── 医疗救护组：伤员急救、转运送医
├── 警戒疏散组：现场警戒、人员疏散
├── 后勤保障组：物资调配、交通保障
└── 事故调查组：原因分析、记录取证
```

## 2. 分类应急处置程序

### 2.1 触电事故

| 步骤 | 处置要点 |
|------|---------|
| 1. 断电 | **立即**切断电源；无法断电时用绝缘物（干木棒/橡胶手套）使伤者脱离带电体 |
| 2. 急救 | 检查呼吸和心跳；无呼吸/心跳立即进行心肺复苏（CPR） |
| 3. 送医 | 拨打 120，途中持续急救；即使伤者恢复意识也必须送医观察 |
| 4. 保护现场 | 设警戒线，保留事故设备和现场状态 |

**禁止行为**：
- 禁止直接用手接触触电者
- 禁止使用潮湿物品解救
- 禁止在未断电前接近高压触电者（10kV 以上保持 2m 安全距离）

### 2.2 高处坠落

| 步骤 | 处置要点 |
|------|---------|
| 1. 保护伤者 | **不要随意搬动**伤者，尤其怀疑脊柱损伤时 |
| 2. 检查伤情 | 检查意识、呼吸、出血、骨折情况 |
| 3. 止血 | 外伤出血用无菌纱布加压包扎；大动脉出血用止血带（记录时间） |
| 4. 固定 | 怀疑骨折的部位用夹板或硬物固定 |
| 5. 送医 | 拨打 120，用担架水平运送，保持脊柱固定 |

**禁止行为**：
- 禁止摇晃或拖拽伤者
- 禁止对骨折部位强行复位
- 禁止让怀疑脊柱损伤的伤者坐起

### 2.3 坍塌事故

| 步骤 | 处置要点 |
|------|---------|
| 1. 撤离 | 立即组织非被困人员撤离危险区域 |
| 2. 警戒 | 划定危险区域，设警戒线，禁止无关人员进入 |
| 3. 搜救 | 确认被困人数和大致位置；使用小型设备谨慎搜救，**禁止大型机械直接作业**以防二次伤害 |
| 4. 急救 | 被困人员救出后检查伤情，进行初步急救 |
| 5. 支护 | 对坍塌区域进行临时支护，防止二次坍塌 |

### 2.4 火灾事故

| 步骤 | 处置要点 |
|------|---------|
| 1. 报警 | 拨打 119；同时启动现场应急 |
| 2. 断电 | 切断火场及相邻区域电源 |
| 3. 灭火 | 初起火灾用灭火器（电气火灾用干粉/CO₂，油类火灾用泡沫/干粉） |
| 4. 疏散 | 组织人员有序撤离，清点人数 |
| 5. 隔离 | 转移周围可燃物和危险品 |

**变压器油火灾特殊处置**：
- 使用干粉灭火器或泡沫灭火器
- 禁止用水直接扑灭（油可能飞溅扩散）
- 开启事故油池排油阀（如有），降低油位
- 关闭冷却系统防止油泵继续运行

### 2.5 物体打击

| 步骤 | 处置要点 |
|------|---------|
| 1. 停止作业 | 立即停止上方施工 |
| 2. 急救 | 检查伤者意识和伤情，止血包扎 |
| 3. 送医 | 头部受伤者即使意识清醒也必须送医检查 |
| 4. 排查 | 检查吊具、脚手架、防护网等安全设施 |

### 2.6 中毒窒息（有限空间）

| 步骤 | 处置要点 |
|------|---------|
| 1. 报警 | 启动有限空间应急响应 |
| 2. 救援 | 救援人员**必须佩戴正压式空气呼吸器**进入；使用三脚架+安全绳提升被困人员 |
| 3. 脱离 | 将伤者转移至空气新鲜处 |
| 4. 急救 | 检查呼吸和心跳，必要时 CPR；吸入有毒气体者给氧气吸入 |
| 5. 送医 | 立即送医，告知医生接触的气体种类 |

**严禁行为**：
- **严禁未佩戴呼吸器盲目进入施救**——这是有限空间事故中造成二次伤亡的首要原因
- 严禁仅依赖口罩进入有毒环境
- 严禁向有限空间内充入纯氧

### 2.7 机械伤害

| 步骤 | 处置要点 |
|------|---------|
| 1. 停机 | 立即停止机械运行，按急停按钮 |
| 2. 解救 | 如肢体被卡住，需拆卸部件解救，**不可反转机器** |
| 3. 急救 | 止血包扎，断肢保存于干净塑料袋内冷藏 |
| 4. 送医 | 拨打 120，说明伤情和断肢保存情况 |

### 2.8 起重伤害

| 步骤 | 处置要点 |
|------|---------|
| 1. 停止作业 | 起重机停止运行，重物就地放置（禁止悬停） |
| 2. 警戒 | 扩大警戒范围，撤离无关人员 |
| 3. 评估 | 如吊具仍挂重物且有倾覆风险，由专业人员评估后再行处置 |
| 4. 急救 | 对伤者进行急救处理 |

## 3. 应急物资清单（通用模板）

| 序号 | 物资名称 | 数量 | 存放位置 | 备注 |
|------|---------|------|---------|------|
| 1 | 急救药箱 | ≥2 个 | 现场办公室 + 施工区 | 含绷带/纱布/碘伏/止血带等 |
| 2 | 担架 | ≥1 副 | 施工区 | 脊柱板型 |
| 3 | 灭火器（干粉 4kg） | ≥6 具 | 配电箱/油区/焊接区/仓库 | 每月检查压力 |
| 4 | 灭火器（CO₂ 5kg） | ≥2 具 | 配电室/电气作业区 | 电气火灾专用 |
| 5 | 正压式空气呼吸器 | ≥2 套 | 有限空间作业区 | 气瓶压力 ≥25MPa |
| 6 | 三脚架救援系统 | 1 套 | 有限空间作业区 | 含安全绳、全身式安全带 |
| 7 | 气体检测仪 | ≥1 台 | 有限空间作业区 | 四合一（O₂/CO/H₂S/可燃） |
| 8 | 安全帽/安全带备用 | 若干 | 材料库 | — |
| 9 | 对讲机 | ≥4 台 | 项目部 | 确保通信畅通 |
| 10 | 应急照明灯 | ≥4 盏 | 施工区 + 生活区 | 防爆型 |
| 11 | 警戒带/警示锥 | 若干 | 材料库 | — |
| 12 | 沙袋/防汛物资 | 若干 | 指定堆放点 | 雨季/汛期必备 |

## 4. 应急演练要求

| 项目 | 要求 |
|------|------|
| 演练频率 | 开工前 1 次（综合演练）+ 施工期间至少每季度 1 次（专项演练） |
| 综合演练 | 覆盖火灾、触电、坍塌等主要事故类型 |
| 专项演练 | 针对本工程高风险作业（如有限空间、起重吊装） |
| 演练评估 | 演练后评估响应时间、人员熟练度，形成演练报告 |
| 预案修订 | 演练暴露的问题纳入预案修订 |
