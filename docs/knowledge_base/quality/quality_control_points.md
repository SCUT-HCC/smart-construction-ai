## 1. 质量管理通用三阶段（事前—事中—事后）

### 1.1 事前控制

| 序号 | 措施内容 | 适用范围 |
|------|---------|---------|
| 1 | 建立健全质量管理组织机构，明确各级人员质量责任 | 通用 |
| 2 | 组织施工人员职业岗位培训，特殊工种持证上岗 | 通用 |
| 3 | 依据审批的施工方案进行质量控制交底，做好交底记录 | 通用 |
| 4 | 进行施工图纸会审和技术交底，做好记录 | 通用 |
| 5 | 收集本工程适用技术标准，确保齐全有效，针对强制性条文编制实施计划 | 通用 |
| 6 | 原材料、构件、半成品建立进场检查验收和取样送检制度，杜绝不合格材料进场 | 通用 |
| 7 | 检测、测量仪器在检定有效期内使用，工器具工况良好 | 通用 |

### 1.2 事中控制

| 序号 | 措施内容 | 适用范围 |
|------|---------|---------|
| 1 | 严格按设计文件、规程规范、工艺标准施工 | 通用 |
| 2 | 坚持质量"三检"制度：自检、互检、上下道工序交接检 | 通用 |
| 3 | 严格控制工序质量，设置 W/H/S 点进行质量控制 | 通用 |
| 4 | 质量记录与工程进度同步 | 通用 |
| 5 | 把好隐蔽工程检查验收关，先验收后隐蔽 | 通用 |
| 6 | 针对质量通病或易产生不合格产品的工序，提前制定措施重点控制 | 通用 |

### 1.3 事后控制

| 序号 | 措施内容 | 适用范围 |
|------|---------|---------|
| 1 | 加强质量验收，严格按标准进行质量评定 | 通用 |
| 2 | 做好成品保护，零缺陷移交 | 通用 |
| 3 | 及时整理质量记录，及时移交竣工资料 | 通用 |
| 4 | 做好施工质量总结和质量回访 | 通用 |

> 来源：DOC 6 第 7.3.1~7.3.3 节

## 2. 变电土建质量通病与防治

| 序号 | 质量薄弱环节 | 预防技术措施 |
|------|------------|------------|
| 1 | 天然地基承载力不够 | ①机械开挖预留人工开挖厚度，避免超挖；②基坑开挖至设计标高后保护坑底，避免积水浸泡；③验收合格后及时浇筑垫层封底 |
| 2 | 混凝土结构蜂窝、孔洞、夹渣 | ①模板支撑稳定不变形，表面平整光滑，接缝密实不漏浆；②原材料计量准确，严格控制坍落度；③分层振捣密实，不漏振不少振不过振；④覆盖浇水养护及时 |
| 3 | 装饰工程效果不佳 | ①由有经验技术熟练工人施工；②加强装饰材料现场验收；③统一规划 |
| 4 | 屋面渗漏 | ①防水卷材按设计和说明书施工，配套使用粘结剂；②细部构造认真处理粘贴牢固；③基层平整按设计坡度施工；④确保搭接宽度和粘贴质量 |
| 5 | 外墙面及窗台渗水 | ①砌筑外墙砖缝砂浆饱满；②墙面抹灰基层清理并淋水湿润；③窗台做外排水坡度，设滴水线；④窗框间隙用水泥砂浆填塞，留槽嵌密封材料 |
| 6 | 预埋件/预留孔遗漏或偏差 | ①设专人预埋并专人检查；②重点注意材料、位置、形状、数量、尺寸、平整度；③可先预埋锚筋后二次埋设 |

> 来源：DOC 6 第 7.3 节"建筑工程保证施工质量对策表"

## 3. 变电土建成品保护要点

| 分项工程 | 保护要点 |
|---------|---------|
| 混凝土工程 | 柱角以木条板保护（高 1.8m）；严禁在混凝土面乱写乱划；薄膜包裹保护 |
| 门窗工程 | 入库堆放整齐，保护膜封闭完整；安装后木板条捆绑防碰撞；保护膜破损需重新粘贴 |
| 屋面工程 | 施工人员穿不带钉子的鞋；卷材上不放金属器具；及时做保护层；清理天沟雨水口 |
| 抹灰工程 | 抹灰前保护预埋件/槽盒/孔洞；硬化前防撞击振动；窗台钉木板条保护 |
| 墙面砖工程 | 1.8m 高度内钉木板保护条；电焊需有遮挡措施防火花 |

> 来源：DOC 6 第 7.4 节

## 4. 变电电气质量控制要点

### 4.1 主变安装关键质量控制点

| 工序 | W/H/S 点 | 质量控制要点 |
|------|---------|------------|
| 本体检查 | W | 运输密封完好性，油样化验合格，绝缘电阻合格 |
| 附件安装 | H | 密封垫压缩量 1/3，螺栓对角线均匀紧固，力矩符合规定 |
| 抽真空 | W | 残压 ≤13Pa，保持时间 ≥24h（500kV），箱壁变形监视 |
| 真空注油 | W | 注油速度 ≤100L/min，温度 40~60℃，全程保持真空 |
| 热油循环 | H | 出口温度 60~70℃，循环 ≥48h，油量 ≥3 倍总油量 |
| 整体密封 | S | 0.03MPa 持续 24h，无渗漏无变形 |
| 电气试验 | S | 油务指标全部达到 GB 50150-2016 要求 |

> W=见证点, H=停工待检点, S=停止点

### 4.2 建筑电气关键质量控制点

| 工序 | 质量控制要点 |
|------|------------|
| 管路敷设 | 暗敷保护层 ≥15mm；弯曲半径 ≥6 倍管径；弯扁度 ≤10% |
| 穿线接线 | 相色标识正确（黄/绿/红/蓝/黄绿）；绝缘电阻 ≥0.5MΩ |
| 配电箱 | 壁挂底边距地 1.5m；垂直度 ≤1.5‰；PE 线严禁串联 |
| 接地装置 | 扁钢搭接 ≥2 倍宽度三面施焊；圆钢搭接 ≥6 倍直径双面焊；焊接处涂防腐 |

## 5. 线路塔基质量控制要点

| 工序 | 质量控制要点 |
|------|------------|
| 测量放线 | 仪器检定有效期内，两人以上工作并复核 |
| 护壁施工 | 每节 ≤1m，混凝土 ≥C25，上下节搭接 ≥50mm |
| 钢筋笼安装 | 分段制作焊接，加劲架间距 3~4m，孔底清理干净才能下放 |
| 混凝土浇筑 | 落距 >3m 用串筒，串筒末端距底 ≤2m；积水多时用导管法 |
| 基础检测 | 桩位偏差、桩径、桩长、垂直度、承载力检测 |

## 6. 特殊作业质量控制要点

### 6.1 防腐涂装

| 工序 | 质量控制要点 |
|------|------------|
| 表面处理 | 除锈等级 Sa2.5，4h 内涂底漆，湿度 >85% 或钢温低于露点 3℃ 禁止施工 |
| 底漆 | 环氧富锌底漆 ≥50μm |
| 中间漆 | 环氧云铁中间漆 ≥100μm |
| 面漆 | 聚氨酯/氟碳面漆 ≥50μm |
| 总厚度 | ≥200μm（室外钢结构 ≥150μm） |

### 6.2 防火涂料

| 工序 | 质量控制要点 |
|------|------------|
| 基层处理 | 防锈底漆涂装完毕并验收合格 |
| 薄型涂层 | 每层 ≤3mm |
| 厚型涂层 | 每层 5~8mm，裂纹宽度 ≤1mm |
| 验收标准 | 无空鼓、无脱落，厚度满足设计耐火极限 |
