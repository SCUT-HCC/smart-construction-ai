# 领域模型

## 施工方案是什么

施工方案是指导施工现场作业的技术文件。核心作用：描述**用什么方法**、**按什么顺序**、**由谁**来完成一个工程，以及**质量、安全、应急**怎么保障。

南方电网的施工方案主要服务于**输变电工程**（变电站新建/扩建、线路架设等）。

## 10 章节结构

基于 16 份实际文档分析，采用扩展的 10 章节结构（原标准 GB/T 50502-2009 为 7 章，实际文档中安全、质量、组织机构等内容占比极高，需独立成章）：

| 章 | 名称 | 核心问题 | 必选 |
|----|------|---------|------|
| 一 | 编制依据 | 这个方案依据什么法规/标准编写的？ | ✅ |
| 二 | 工程概况 | 这是什么工程？在哪？多大规模？ | ✅ |
| 三 | 施工组织机构及职责 | 谁来干？各自干什么？ | ✅ |
| 四 | 施工安排与进度计划 | 先干什么后干什么？多久完成？ | ✅ |
| 五 | 施工准备 | 开工前要准备什么？人/材/机/场地 | ✅ |
| 六 | 施工方法及工艺要求 | 具体怎么干？ | ✅ |
| 七 | 质量管理与控制措施 | 怎么保证干得好？ | ✅ |
| 八 | 安全文明施工管理 | 怎么保证不出事？ | ✅ |
| 九 | 应急预案与处置措施 | 出了事怎么办？ | ✅ |
| 十 | 绿色施工与环境保护 | 怎么减少环境影响？ | ⭐ 可选 |

详细子章节定义见 [03-chapter-specification](03-chapter-specification.md)。

## 文档类型分类

16 份样本涵盖多种施工方案类型：

| 类型 | 文档编号 | 特点 |
|------|---------|------|
| **综合施工组织设计** | 1, 7, 9, 10, 11, 15 | 覆盖章节最全，结构最接近标准 |
| **专项施工方案** | 2(喷涂), 3(防火), 4(钢屋面), 6(主控楼), 12(灌注桩), 14(装修) | 聚焦特定工序，章节结构因工序而异 |
| **专题方案** | 5(绿色施工), 8(雨季), 13(塔吊), 16(安健环) | 专注于某一管理维度（季节性/专题性） |

**关键发现**：专项施工方案的第六章（施工方法及工艺要求）差异极大，与工程类型强相关。其他章节（编制依据、安全、质量、应急）结构相对稳定。

## 术语表

| 术语 | 含义 |
|------|------|
| **编制依据** | 方案编写所参照的法规、标准、设计文件、合同 |
| **工序** | 施工过程中的一个独立作业步骤 |
| **危险源** | 可能导致人身伤害或财产损失的因素 |
| **质量通病** | 施工中反复出现的常见质量问题 |
| **三通一平** | 通水、通电、通路、场地平整——开工前的基本条件 |
| **四节一环保** | 节能、节地、节水、节材 + 环境保护 |
| **安健环** | 安全、健康、环保的合称 |
| **施工组织设计** | 比施工方案更上层的文件，覆盖整个项目的组织和规划 |
| **专项施工方案** | 针对特定工序（如吊装、基坑、高处作业）的专用方案 |
| **GB/T 50502-2009** | 《建筑施工组织设计规范》，定义标准 7 章结构 |
