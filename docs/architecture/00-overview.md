# 南网施工方案智能辅助系统 — 总纲

> 为南方电网构建施工方案的智能检测（审核）与生成系统。

## 项目目标

减轻施工方案编制和审核的人力投入。两大核心功能：

1. **施工方案检测（审核）** — 输入一份施工方案，自动检查章节完整性、编制依据时效性、内容合规性，输出检测报告
2. **施工方案生成** — 输入工程基本信息，基于知识库和优质样本，自动生成完整的 10 章节施工方案

## 技术栈

| 层级 | 选择 |
|------|------|
| 语言 | Python 3.10 (conda: sca) |
| LLM 框架 | LangChain + LangGraph |
