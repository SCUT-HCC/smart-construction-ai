# 章节规格书

> 每一章必须写什么、质量标准是什么、内容从哪里来。
> 命名规范详见 `templates/naming_conventions.md`。

---

## 一、编制依据

**核心问题**：这个方案依据什么编写的？

### 必写内容

| 子章节 | 内容 | 来源 |
|--------|------|------|
| 1.1 法律法规 | 适用的国家法律法规 | 知识库（规范库） |
| 1.2 行业标准 | 国标(GB)、行标(DL)、企标(Q/CSG) | 知识库（规范库）+ 工程类型匹配 |
| 1.3 设计文件 | 工程设计图纸、设计说明 | 用户输入 |
| 1.4 合同约定 | 施工合同相关条款 | 用户输入 |

### 质量判据

- [ ] 引用的标准必须是**现行有效版本**（审核系统的时效性检查核心）
- [ ] 标准编号格式正确（如 GB 50300-2013，不能缺少年份）
- [ ] 覆盖安全、质量、环保三个维度的标准
- [ ] 与工程类型匹配（输变电工程应引 DL/T 系列标准）

### 内容来源

- **标准列表** → 知识库检索（按工程类型）
- **设计文件/合同** → 用户输入（工程参数）
- **法规** → 通用模板（变化少）

## 二、工程概况

**核心问题**：这是什么工程？

### 必写内容

| 子章节 | 内容 | 来源 |
|--------|------|------|
| 2.1 工程简介 | 名称、地点、性质、建设内容 | 用户输入 |
| 2.2 工程规模 | 工程量、主要技术参数 | 用户输入 |
| 2.3 工程地质及地貌 | 地形、地质条件、气候 | 用户输入 + 知识库（地区气候模板） |
| 2.4 参建单位 | 建设/设计/施工/监理单位 | 用户输入 |

### 质量判据

- [ ] 工程名称全文一致（审核系统的事实一致性检查核心）
- [ ] 技术参数完整（电压等级、容量、长度等关键参数不缺）
- [ ] 地质条件与后续施工方法匹配（如软土地基 → 应有地基处理方法）

### 内容来源

主要靠**用户输入**。这一章基本是填参数，LLM 负责组织语言。

## 三、施工组织机构及职责

**核心问题**：谁来干？

### 必写内容

| 子章节 | 内容 | 来源 |
|--------|------|------|
| 3.1 项目组织架构 | 架构图（项目经理 → 各部门） | 模板 + 用户输入 |
| 3.2 管理人员职责 | 项目经理、总工、部门负责人的职责 | 知识库（通用职责模板） |
| 3.3 质量安全人员配置 | 质量员、安全员、技术员 | 知识库 + 工程规模匹配 |

### 质量判据

- [ ] 组织架构完整（项目经理、总工、安全/质量/技术负责人必须有）
- [ ] 职责描述与工程类型匹配
- [ ] 人员配置与工程规模匹配（大型工程不能只配 2 个安全员）

### 内容来源

职责描述高度模板化（知识库），人员数量需根据工程规模调整。

## 四、施工安排与进度计划

**核心问题**：先干什么后干什么？

### 必写内容

| 子章节 | 内容 | 来源 |
|--------|------|------|
| 4.1 施工总体部署 | 施工阶段划分、流水段划分 | 案例检索 + LLM 生成 |
| 4.2 施工顺序安排 | 各工序的先后关系 | 工程类型模板 + 案例 |
| 4.3 施工进度计划 | 时间节点、甘特图 | 用户输入（工期）+ LLM 排布 |
| 4.4 关键节点与里程碑 | 关键验收节点 | 案例检索 + 规范要求 |

### 质量判据

- [ ] 工期与工程规模匹配（不能不合理的短/长）
- [ ] 工序顺序逻辑正确（不能先砌墙后做基础）
- [ ] 关键节点明确

### 内容来源

框架靠案例检索，具体时间由用户输入 + LLM 调整。

## 九、应急预案与处置措施

**核心问题**：出了事怎么办？

### 必写内容

| 子章节 | 内容 | 来源 |
|--------|------|------|
| 9.1 应急组织机构与职责 | 架构图、职责分工 | 通用模板 |
| 9.2 应急响应程序 | 报告流程、响应级别 | 通用模板 |

| 9.5 应急演练计划 | 演练频次、内容 | 通用模板 |

### 质量判据

- [ ] 应急类型覆盖该工程的所有风险（与第八章的危险源对应）
- [ ] 每种事故有具体的处置措施（不是空洞的"立即报告"）
- [ ] 物资清单完整

### 内容来源

高度模板化。应急处置措施按事故类型（触电/坍塌/高坠/火灾/物体打击）有标准写法，从知识库检索。

## 章节间依赖关系

```
第二章（工程概况）──→ 第四章（施工安排：基于工程规模排工期）
                  ──→ 第五章（施工准备：基于工程量定设备/材料）
                  ──→ 第六章（施工方法：基于地质条件选工艺）

第六章（施工方法）──→ 第七章（质量管理：控制点对应工序）
                  ──→ 第八章（安全管理：危险源对应工序）

第八章（安全管理）──→ 第九章（应急预案：应急类型对应风险）
```

生成系统**必须串行生成**，后续章节依赖前序章节的内容。
