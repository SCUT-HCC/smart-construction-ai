## 章节覆盖率分析

将 16 篇文档的实际章节映射到 10 章标准结构后：

| 标准章节 | 覆盖率 | 说明 |
|---------|--------|------|
| 一、编制依据 | 56.2%（9/16） | 部分文档用"编制说明"替代 |
| 二、工程概况 | 68.8%（11/16） | 覆盖率最高，命名变体多（概况/概述/简介） |
| 三、施工组织机构及职责 | ~50%（分散） | 几乎都有但命名极不统一，常散在不同位置 |
| 四、施工安排与进度计划 | 12.5%（2/16） | 独立成章的少，常并入其他章节 |
| 五、施工准备 | 18.8%（3/16） | 同上 |
| 六、施工方法及工艺要求 | 高（分散） | 每份都有核心工艺内容，但命名和结构完全不同 |
| 七、质量管理与控制措施 | 31.2%（5/16） | "质量工艺要求"是最常见的变体 |
| 八、安全文明施工管理 | 25%+（4/16） | 加上安健环、文明施工等变体覆盖更高 |
| 九、应急预案与处置措施 | 25%（4/16） | 几乎都有应急内容但独立成章的少 |
| 十、绿色施工与环境保护 | 18.8%（3/16） | 趋势性章节，DOC 5 是专题方案 |

**核心发现**：

1. **没有一个章节在所有文档中出现**——最高的"工程概况"也只有 68.8%
2. **章节命名极度不统一**——同一内容有 3~5 种命名变体
3. **结构因方案类型而异**——综合施工组织设计（DOC 1/7/9/10/11/15）结构最接近标准；专项方案结构差异大
4. **安全相关内容占比极高**——安全管理、危险源分析、应急处置在实际文档中远比标准 7 章的"施工保证措施"丰富

## 跨文档共性模式

**结构稳定的章节**（跨文档写法相似，适合模板化）：
- 编制依据：法规 + 标准 + 设计文件 + 合同，格式固定
- 工程概况：工程简介 + 规模 + 地质 + 参建单位，基本都是填参数
- 施工组织机构：架构图 + 职责表，模式化
- 应急预案：组织机构 + 响应程序 + 处置措施（按事故类型），模式化

**结构不稳定的章节**（跨文档差异大，需要分类处理）：
- 施工方法及工艺要求：完全取决于具体工序（喷涂/浇筑/吊装/强夯/...），无法通用模板化
- 质量管理：内容深度差异大，有的只有一页，有的几十页
- 施工安排：有的按工序排，有的按时间排，格式不统一

## 工程类型与章节结构的关系

| 工程类型 | 文档 | 第六章特殊性 |
|---------|------|------------|
| 基础工程（基坑/灌注桩） | 1, 12, 15 | 开挖 → 扎筋 → 浇筑 → 养护 → 回填 |
| 安装工程（变压器/钢结构） | 7, 10 | 设备就位 → 附件安装 → 调试 → 验收 |
| 建筑工程（楼/装修） | 4, 6, 14 | 主体结构 → 砌筑 → 抹灰 → 装饰 |
| 涂装工程（防火/喷涂） | 2, 3 | 表面处理 → 调配 → 施工 → 验收 |
| 专题方案 | 5, 8, 13, 16 | 无统一模式 |

**结论**：第六章必须按工程类型分模板。其他章节可以复用通用模板。
