# 南方电网施工方案参考库 - 特殊作业与通用管理篇

本文件汇集了雨季/防台防汛、有限空间作业、绿色施工及环境管理等通用性较强的标准化管理措施。Agent在编写相关章节时，可直接引用以下段落。

## 1. 雨季施工及防台防汛措施
### 1.1 雨季施工原则
1.  **防雨原则**：小雨不停工，大雨大风停停止室外施工。
2.  **排水原则**：施工场地必须做好排水沟规划，规划排水分区，保证大雨来临不致雨水灌至基坑。
3.  **防护原则**：机械设备露天作业的，要搭设防晒、防雨、防砸等安全防护棚。特殊工种的操作人员必须持证上岗。施工前班组长必须负责对机械设备操作、维修人员进行技术交底。

### 1.2 混凝土工程雨季措施
1.  **浇筑控制**：雨施季节浇筑前，检查脚手架和脚手板是否牢固，防滑措施是否可靠。在浇筑前应及时准确地掌握天气情况，小雨天气可浇筑砼，尽量避开雨天施工。
2.  **坍落度调整**：雨期施工期间，砂石含水率变化较大，要求混凝土供应商及时测定砂石含水率，并调整混凝土施工配合比，严格控制混凝土的坍落度，确保混凝土的强度。
3.  **覆盖保护**：掌握天气变化，做好天气预报，调整工作时间合理预留混凝土缝，准备防雨台布，混凝土浇筑未凝时，严禁使表面雨水冲刷，要保证现场有足够的覆盖材料，及时覆盖。

### 1.3 钢筋及焊接雨季措施
1.  **防锈蚀**：钢筋堆放场地应平整，排水通畅，不得有大面积积水现象，雨季来临前应对场地进行平整。钢筋按不同规格分批堆放，分不开堆，不得混杂，钢筋下放置垫木，下雨前钢筋用塑料布覆盖，雨后及时晾晒通风，减少钢筋锈蚀。
2.  **焊接要求**：焊接设备注意防潮、防雨和防高温。雨天不得在无任何防雨措施的场地进行焊接施工。焊接接头焊后不能立即接触雨水。
3.  **焊条管理**：焊条受潮后使用前必须进行烘干。碱性低氢型药皮电焊条使用前必须进行 300～350℃烘焙，酸性药皮焊条在夏季使用前也应进行 100～150℃烘焙。烘焙后的焊条要储存在 100℃左右的恒温箱内随用随取。

### 1.4 临时用电与机具防护
1.  **配电箱防护**：现场施工用高低压设备及线路，按规程进行安装和架设。严禁使用破损或绝缘性能不良的电线，严禁电线随地走，满地爬，所有电闸箱应上锁，有防雨盖板，有危险标志。
2.  **接地保护**：各种用电机械、机具都装有灵敏有效的漏电保护装置，接地接零要规范。实行分级配电，实行“一机、一闸、一漏、一箱”，各分部统一按规定设置配电、开关箱。
3.  **照明安全**：布置足够的照明设备，保证施工照明强度要求，确保施工现场的安全，防止夜间行走踩空和车辆相撞等事件。基坑内和基坑边必须配备足够照明灯，并设围栏和安全标识。

## 2. 有限空间作业安全措施

### 2.1 作业原则与流程
必须严格执行“先通风、再检测、后作业”的原则。①未经风险辨识不准作业；②未经通风和检测合格不准作业；③不佩戴劳动防护用品不准作业；④没有监护不准作业；⑤电气设备不符合规定不准作业；⑥未经审批不准作业；⑦未经培训演练不准作业。

### 2.2 通风要求
1.  **强制通风**：地下有限空间范围通风条件复杂时，应进行通风设计并经作业单位审批后作业。机械通风应满足下列要求：①作业区横断面平均风速不小于 0.8m/s 或通风换气次数不小于 20 次/h；②地下有限空间只有一个出入口时，将通风设备出风口置于作业区区域，进行送风作业。
2.  **设备布置**：地下有限空间有两个或两个以上出入口、通风口时，临近作业者处进行送风，远离作业者处进行排风。必要时，可设置挡板或改变吹风方向以防止出现死角。

### 2.3 气体检测要求
1.  **检测指标**：检测指标包括氧浓度值、易燃易爆物质（可燃性气体、爆炸性粉尘）浓度值、有毒气体浓度值等。最低限度应检测下列三项：氧浓度（应在 19.5-23.5%范围内），易燃/可燃气体浓度（应 <最低爆炸极限的 10%），一氧化碳浓度（应<25ppm）。
2.  **检测频率**：在作业环境条件可能发生变化时，应对作业场所中危害因素进行持续或定时检测。实施检测时，检测人员应处于安全环境，检测时要做好检测记录，包括检测时间、地点、气体种类和检测浓度等。
3.  **检测点位**：本工程有限空间纵向高度均为 8m 以下，按照规范检测点的数量应不少于 2 个；上下检测点，距离地下有限空间顶部和底部均不应超过 1m。

### 2.4 现场监护与应急
1.  **专人监护**：作业现场必须配置专职监护人员，监护人员必须保持与井下作业人员的联系，并不得擅自离开岗位。
2.  **应急装备**：现场必须配置三脚架、安全绳（全身式安全带）、正压式空气呼吸器、气体检测仪、防爆对讲机及急救箱。
3.  **紧急撤离**：作业过程中，一旦检测仪报警或作业人员感到不适，必须立即停止作业，撤离有限空间。

## 3. 绿色施工与环境保护

### 3.1 扬尘控制 ("六个100%")
1.  **围挡作业**：施工现场必须设置连续、封闭的围挡，围挡高度不低于 2.5m（市区主要路段）或 1.8m（一般路段），做到坚固、平稳、整洁、美观。
2.  **裸土覆盖**：施工现场裸露的场地和堆放的土方应采取覆盖、固化或绿化等措施。超过 48 小时不施工的裸土必须进行覆盖。
3.  **车辆冲洗**：施工现场出入口必须设置车辆冲洗设施，建立冲洗台账，保证车辆净车出场。冲洗台周边设置排水沟和沉淀池。
4.  **洒水降尘**：施工现场土方作业、拆除作业等易产生扬尘的工序，必须采取洒水、喷雾等降尘措施。遇到四级以上大风天气，不得进行土方回填、转运以及其他可能产生扬尘污染的施工。

### 3.2 噪音与光污染控制
1.  **时段控制**：严格控制作业时间，夜间（22:00 至次日 6:00）禁止进行产生环境噪声污染的建筑施工作业。因特殊需要必须连续作业的，必须办理夜间施工许可证并公告附近居民。
2.  **设备降噪**：尽量选用低噪声设备，对产生噪音的机械设备（如发电机、空压机）搭建防噪棚。
3.  **光污染**：夜间室外照明灯加设灯罩，透光方向集中在施工范围。电焊作业采取遮挡措施，避免电焊弧光外泄。

### 3.3 水资源利用与排放
1.  **循环用水**：现场设置循环水系统，基坑降水、雨水收集后用于降尘、车辆冲洗和绿化灌溉。
2.  **污水排放**：施工现场厕所设置化粪池，食堂设置隔油池。生产废水（如泥浆水、养护水）必须经沉淀池沉淀达到排放标准后，方可排入市政污水管网。

## 4. 文明施工与标准化

### 4.1 "7S"管理实施
施工现场全面推行“7S”管理（整理、整顿、清扫、清洁、素养、安全、节约）。
*   **整理**：区分要与不要，清除现场无用物品。
*   **整顿**：定置管理，物料摆放整齐有序。
*   **安全**：消除隐患，确保本质安全。

### 4.2 标识标牌标准化
1.  **五牌一图**：在工地大门入口处设置工程概况牌、管理人员名单及监督电话牌、消防保卫牌、安全生产牌、文明施工牌和施工现场平面布置图。
2.  **安全警示**：在孔洞、临边、高压线、吊装区域等危险部位，悬挂符合国标的安全警示标志（禁止、警告、指令、提示）。

### 4.3 现场围蔽
施工现场分区明确，办公区、生活区与作业区应分开设置，并保持安全距离。高压设备区、GIS室等重点区域应设置定型化围栏，并挂设“止步，高压危险”等标示牌。
